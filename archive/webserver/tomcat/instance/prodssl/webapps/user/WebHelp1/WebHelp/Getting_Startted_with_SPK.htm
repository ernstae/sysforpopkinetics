Getting Startted with SPK

The Model Design Agent (MDA) provides access to the SPK software. The MDA Input File Generation Tool creates a suitable input file that, along with a data file, are all that is needed to submit a job to SPK.

 

After logging into MySPK , click on Get my Model Design Agent and follow instructions for downloading the MDA. The MDA application is downloaded anew in each session to ensure access to the latest development version.

 

Once the MDA is open, click on "Wizard" to open the MDA Input File Generation Tool. The Input File Generation Tool is designed to be self-explanatory, but the following simple example can help speed familiarity with the process.
