1. Analysis Selection.

In the first screen of the Input File Generation Tool (select Wizard),

 

  • choose: "Population" and "Parameter estimation" as the analysis type, since we have a population of rails we wish to understand, as opposed to a single individual rail.; and
  • choose: "Parametric" as the statistical modeling type, since that is what we are attempting to do;
  • check: "Analytic/algebraic model" as opposed to using an ODE model or pre-defined model from the library.
  • Thinking ahead, we also checked options for "Computing statistics of parameter estimates", "Tables" and "Scatterplots" so that screens for defining these options will be available later.

 

 

Click Next> to continue with Input generation. You may return to any screen at any time (by clicking the <Back button) without loosing information. Clicking "Help" from any page of the Input File Generation Tool opens the MDA Help File to the appropriate page.

(The "Load Model or Input" button allows the user to reload and edit a previously saved job.)
