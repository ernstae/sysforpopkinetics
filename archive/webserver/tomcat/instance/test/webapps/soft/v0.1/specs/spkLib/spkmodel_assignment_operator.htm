Assignment Operator
Prototype: protected: template<class Scalar> SpkModel<Scalar>::SpkModel& operator=( const SpkModel &right )

Description
The assignment operator.

Since SpkModel is an abstract class, the direct use of this operator is only allowed for its subclasses.

The assignment operator is provide to make the SpkModel class more standard even though SPK itself does not currently require its model classes to be assignable.

Arguments
right
The SpkModel to be copied.
