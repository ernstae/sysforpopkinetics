Element <sigma>

A minimal representation of NONMEM Omega matrix when optimization completes (ie. converges) successfully. If fails, this entry contains the value from the last successfully completed iteration.

For full (ie. "block") matrix, sigma lists the lower triangle elements in the row-major order. For diagonal, sigma lists only the diagonal elements from the upper left corner. Given a 3 by 3 matrix, A: 

/                   \
|  a11   a12   a13  |
|  a21   a22   a23  |
|  a31   a32   a33  |
\                   /
If A were full (ie. block) symmetric, sigma specifies as follows: 
    <sigma>
       <in>
          <value>a11</value>
          <value>a21</value><value>a22</value>
          <value>a31</value><value>a32</value><value>a33</value>
       </in>
    </sigma>
If A were diagonal, sigma specifies as follows: 
    <sigma>
       <in>
          <value>a11</value>
                           <value>a22</value>
                                             <value>a33</value>
       </in>
    </sigma>
