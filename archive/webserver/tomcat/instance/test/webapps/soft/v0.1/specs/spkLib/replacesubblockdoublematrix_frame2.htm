Description
Replaces a sub-block of a matrix A by another matrix B.

Arguments

& A
mA by nA arbitrary matrix.

& B
mB by nB matrix (mB < mA and nB < nA).

startRow
specifies the starting row index (0 < startRow < (mA - mB)).

startCol
specifies the starting column index (0 < startCol < (nA - nB)).

Example
If you compile, link, and run the following program:

    #include "replaceSubblock.h"

    void main()
    {
        DoubleMatrix dmatA(3,3);
        DoubleMatrix dmatB(2,2);
        double *pdmatA = dmatA.data();
        double *pdmatB = dmatB.data();

        // set the matrix A to:
        // 
        // [0, 3, 6]
        // [1, 4, 7]
        // [2, 5, 8]
        //
        for( int i=0; i<3*3; i++ )
            pdmatA[i] = i;

        // set the matrix B to:
        // 
        // [4, 2]
        // [3, 1]
        //
        for( i=0; i<2*2; i++ )
            pdmatB[i] = 4 - i;

        replaceSubblock(dmatA, 0, 0, dmatB);
        dmatA.print();
    }

 then it will display the following when it is run:

      [4, 2, 6]
      [3, 1, 7]
      [2, 5, 8]
