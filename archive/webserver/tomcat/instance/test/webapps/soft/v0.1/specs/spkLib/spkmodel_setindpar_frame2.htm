Description
Required for both population analysis and individual analysis.

SpkModel<Scalar>::doSetIndPar( const SPK_VA::valarray<Scalar>&  indPar ) sets indPar as the current individual parameter value and guarantees that subsequent attempts for evaluating the mathematical models expressed by SpkModel virtual member functions are evaluated at the point.

doSetPopPar() shall be only called via its corresponding public interface, setIndPar(). The public interface shall be called only by the SPK library components unless when it is being unit-tested, or SPK cannot guarantee any of the computational results.

The public interface, setIndPar() catches any exception thrown by the private interface, doSetIndPar(), implemented by the user. If the exception is an instance of SpkException , an additional SpkError object will be appended to the list of errors maintained within the object. The new SpkError object will have error code set to SpkError::SPK_MODEL_SET_IND_ERR.

Arguments
indPar
is an individual parameter vector of size nIndPar, where nIndPar > 0.
