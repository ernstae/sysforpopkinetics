Description
Given a m times n dimensional column vector, rvecInverse() retrieves a m by n matrix A, such that A(i,j) = a(i*m+j).

Arguments

a
is a m times n dimensional column vector.

m
specifies the number of cols in the resulting matrix. It must be greater than zero.

A
is a reference to a m times n matrix that will contain the resulting matrix.

Example
If you compile, link, and run the following program:
    #include <iostream>
    #include "DoubleMatrix.h"
    #include "rvecInverse.h"

    void rvecInverseEx()
    {
        using namespace std;
        const int m = 2;
        const int n = 3;

        // initialize a matrix A
        DoubleMatrix A(m,n);
        for(int i=0; i<m*n; i++)
        {
            A.data()[i] = i+1;
        }

        // 
        DoubleMatrix a = rvec(A);
        DoubleMatrix C = rvecInverse(a,n);
        cout << "A       = " << A << endl;
        cout << "rvec(A) = " << a << endl;
        cout << "rvec^-1(a," << m << ") = " << C << endl;
    }
