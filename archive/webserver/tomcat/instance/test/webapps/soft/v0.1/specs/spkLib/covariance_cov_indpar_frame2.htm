Description
The public interface void cov_indPar( DoubleMatrix& \ret\ ) const evaluates the user defined virtual doCov_indPar( DoubleMatrix& \ret\ ) const at the current evaluation points.

Arguments
ret 
is n by nIndPar matrix resulted from evaluating doCov_indPar(), where n is the product of #rows and #cols of a matrix resulted from evaluating doCov() and nIndPar is the size of the individual parameter vector. j-th column of matrix, ret, contains the derivative of the covariance with respect to j-th element of the individual parameter vector, in the column-major order. Each column, or the derivative with respect to j-th, must be a symmetric positive definite matrix.
