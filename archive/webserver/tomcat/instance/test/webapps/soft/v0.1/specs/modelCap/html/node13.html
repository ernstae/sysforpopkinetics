Modeling Capability Restrictions

SPK will not support all of NONMEM's modeling capabilities.

In particular, the modeling capabilities associated with some of the NM-TRAN records and keywords will be restricted. These restriction are discussed in this section.


Subsections
  • Data Item Types ($INPUT)
    • Reserved Labels not Recognized by SPK
    • Dependent Variable Data Item (DV)
    • Event Identification Data Item (EVID)
    • Identification Data Item (ID)

  • Data Sets ($DATA)
    • Event Identification Data Items (EVID)
    • Steady State Data Items (SS)

  • Model Specification via the PRED Block ($PRED)
    • Variable Naming Restrictions
    • Special NONMEM-PREDPP Arguments (NEWIND and ICALL)
    • Variables from NONMEM-PREDPP Common Blocks
    • Controlling the Calling Frequency (CALLFL)
    • First-order Approximation for the Intraindividual Error Model during Population Level Estimation

  • Pharmacokinetic Model Specification ($SUBROUTINE)
    • Pharmacokinetic Library Subroutines (ADVAN)
    • Translator Routines (TRANS)

  • Pharmacokinetic Parameter Models ($PK)
    • No Support for $PK Block
    • Specifiying Pharmacokinetic Parameter Models using $PRED

  • Initial Estimates and Bounds for Theta ($THETA)
    • Unbounded Upper or Lower Limits

  • Initial Estimates for Omega ($OMEGA)
    • Diagonal or Full Matrices Only
    • Only One Block for Nondiagonal Matrices (BLOCK)

  • Intraindividual Error Models ($ERROR)
    • No Support for $ERROR Block
    • Specifiying Intraindividual Error Models using $PRED

  • Initial Estimates for Sigma ($SIGMA)
    • Diagonal or Full Matrices Only
    • Only One Block for Nondiagonal Matrices (BLOCK)

  • Instructions for the Estimation Step ($ESTIMATION)
    • Continue Estimation when Theta is Nonphysiological (NOABORT)
    • Conditional Likelihood for Noncontinuous Observed Responses (LIKELIHOOD)
    • Constrain Average Eta Estimates to be Close to Zero (CENTERING)
    • Request Second Eta Derivatives to no be Numerical (NONUMERICAL)
    • Fix Selected Etas to be Zero (ZERO)
