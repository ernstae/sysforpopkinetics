Replace a sub-block of a matrix by another matrix [valarray]
Prototype: void replaceSubblock( 
                          valarray<double> &a, 
                          int nACols, 
                          valarray<double> &b, 
                          int nBCols, 
                          int strRow, 
                          int strCol 
                          )
See also: DoubleMatrix version of replaceSubblock()

Copyright (C) 2002, University of Washington, Resource Facility for Population Kinetics. All Rights Reserved.

Description
Replaces a sub-block of a matrix A stored as a vector by another matrix B stored as a vector.

Arguments

& A
mA by nA arbitrary matrix.

& B
mB by nB matrix (mB < mA and nB < nA).

startRow
specifies the starting row index (0 < startRow < (mA - mB)).

startCol
specifies the starting column index (0 < startCol < (nA - nB)).

Example
If you compile, link, and run the following program:

    #include "SpkValarray.h"
    #include "replaceSubblock.h"

    void main()
    {
        valarray<double> a(3 * 3);
        valarray<double> b(2 * 2);

        // matrix A is:
        // 
        // [0, 3, 6]
        // [1, 4, 7]
        // [2, 5, 8]
        //
          //
          // store A in a vector:
          //
          // a = { 0, 1, 2, 3, 4, 5, 6, 7, 8 }
          //
        for( int i=0; i<3*3; i++ )
            a[i] = i;

        // matrix B is:
        // 
        // [4, 2]
        // [3, 1]
        //
          // store B in a vector:
          //
          // b = { 4, 3, 2, 1 }
          //
        for( i=0; i<2*2; i++ )
            b[i] = 4 - i;

        replaceSubblock(a, 3, b, 2, 0, 0);
        DoubleMatrix( a, 3 ).print();
    }

 then it will display the following when it is run:

      [4, 2, 6]
      [3, 1, 7]
      [2, 5, 8]
