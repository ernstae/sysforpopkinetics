Element <link>

Required only in the absence of pk abbrivated code. link defines a flow rate constant, k, from compartment-"from" to compartment-"to". If the flow is bidirectional, specify the flows with two link elements.

Attributes

k REQUIRED

Flow rate constant of the compartment

to REQUIRED

Identification number of the compartment to which the flow move in.

from REQUIRED

Identification number of the compartment from which the flow originates.
