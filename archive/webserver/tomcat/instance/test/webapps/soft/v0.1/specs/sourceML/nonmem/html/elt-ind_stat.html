Element <ind_stat>

Request for the post statistial analysis on the final estimate(s) for individual parameter(s). This process will be skipped unless the "is_estimation" of ind_analysis is set to "yes".

Attributes

is_stderror_out IMPLIED

"yes" to request to return the standard error of the final estimate(s) for individual parameter(s), default to "yes".

is_covariance_out IMPLIED

"yes" to request to return the covariance matrix of the final estimate(s) for individual parameter, default to "yes".

is_correlation_out IMPLIED

"yes" to request to return the correlation matrix of the final estimate(s) of individual parameter, default to "yes".

is_confidence_out IMPLIED

"yes" to request to return the confidence intervals for the final estimate(s) for individual parameter, default to "yes".

covariance_form REQUIRED

Form of covariance matrix; "r" for R^-1, "rsr" for R^-1*S*R^-1, "s" for S^-1, "h" for H^-1 and "hsh" for H^-1*S*H^-1.

is_coefficient_out IMPLIED

"yes" to request to return the coefficents for the final estimate(s) for individual parameter, default to "yes".

is_inverse_covariance_out IMPLIED

"yes" to request to return the inverse of covariance matrix of the final estimate(s) for individual parameter, default to "yes".
