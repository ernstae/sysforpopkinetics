Re-organize Matrix Containing A_x to form A^T_x (DoubleMatrix version)
// // [ Comment by Sachiko, 10/01/02 ] // Taking A as an argument just for the sake of obtaining the dimensions seems // incorrect. I might have done so when this routine was local, which // does not require the same level of standard imposed on public interfaces. // But, as long as it is published as a public, the interface design // should have been thoroughly thought. //
Prototype: const DoubleMatrix transposeDerivative(const DoubleMatrix &A, 
                                               const DoubleMatrix& A_x,)
See also: valarray version .

Copyright (C) 2002, University of Washington, Resource Facility for Population Kinetics. All Rights Reserved.

Description
Returns the derivative of
     A(x)' 
where the prime ' denotes transpose of the matrix.

Arguments
// // [ Comment by Sachiko, 10/01/02 ] // This argument should not be part of the interface // as long as the soul purpose of this value is to // obtain the dimensions of the matrix. // It may force the caller to evaluate the function // to get the value when it is not really needed. // The function should rather ask for the dimensions directly. //
A
is a m by n matrix.
A_x
is a m*n by p matrix that represents a value of a derivative of A(x) with respect to vector x, where p is the dimension of x. // // [ Added by Sachiko, 10/01/02 ] // The elements of each column of A_x, j-th column, are assumed to be arranged in the rvec-ed fashion.
   i.e. A_x = [ rvec(A_x1), rvec(A_x2), ... rvec(A_xp) ]

Example
If you compile, link, and run the following program:
    #include <iostream>
    #include "DoubleMatrix.h"
    #include "transposeDerivative.h"

    void main()
    {
        using namespace std;

        const double x[] = {1.0, 2.0};

        //
        // Set A to:
        //   [ 0     x[1] ]
        //   [ x[0]  1    ]
        //
        DoubleMatrix A(2,2);
        A.data()[0] = 0;
        A.data()[1] = x[0];
        A.data()[2] = x[1];
        A.data()[3] = 1.0;

        //
        // Therefore A_x is:
        //
        //    [ 0  0 ]
        //    [ 0  1 ]
        //    [ 1  0 ]
        //    [ 0  0 ]
        //
        DoubleMatrix A_x(4,2);
        A_x.fill(0);
        A_x.data()[2] = 1.0;
        A_x.data()[5] = 1.0;

        //
        //
        // C = (A')_x should be:
        //
        //   [ 0  0 ]
        //   [ 1  0 ]
        //   [ 0  1 ]
        //   [ 0  0 ]
        //
 
        DoubleMatrix C = transposeDerivative(A, A_x);

        cout << "C =" << endl;
        C.print();
    }
 then it will display the following when it is run:
    
    C =
    [ 0  0 ]
    [ 1  0 ]
    [ 0  1 ]
    [ 0  0 ]
