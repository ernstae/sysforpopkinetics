Description
Required for both population and individual analysis. A default implementation is provided.

SpkModel<Scalar>::doDataVarianceInv_indPar() calculates the derivative of the inverse of the variance of a particular individual's data with respect to the individual parameter, for a currently selected individual (see selectIndividual ) at a currently set individual parameter (see setIndPar ) and population parameter (see setPopPar ).

The evaluation result of doDataVarianceInv_indPar() shall be placed in ret. It returns true if ret contains at least one non-zero value. If all zeros, it returns false.

doDataVarianceInv_indPar() shall be only called via its corresponding public interface, dataVarianceInv_indPar(). The public interface shall be called only by the SPK library components unless when it is being unit-tested, or SPK cannot guarantee any of the computational results.

The public interface, doDataVarianceInv_indPar() catches any exception thrown by the private interface, dataVarianceInv_indPar(), implemented by the user. If the exception is an instance of SpkException , an additional SpkError object will be appended to the list of errors maintained within the object. The new SpkError object will have error code set to SpkError::SPK_MODEL_DATA_VARIANCE_IND_ERR.

Reference
B. M. Bell, Approximating The Marginal Likelihood Estimate For Models With Random Parameters, Applied Mathematics and Computation, Accepted 1999.

Arguments
ret
shall, upon a successful completion of evaluation, contain a resulting vector of size nYi * nYi * nIndPar, where nYi is the number of measurements for a currently selected individual and nIndPar is the size of individual parameter vector. Let RInv denotes the function expressed by doDataVarianceInv(). The resulting vector contains a sequence of partial derivatives of RInv(popPar, indPar) with respect to each element of indPar (individual parameter vector):

  ret = { RInv(popPar, indPar)_indPar(1), RInv(popPar, indPar)_indPar(2), ... RInv(popPar, indPar)_indPar(nIndPar) }

where indPar(k) denotes k-th element of indPar vector. RInv(popPar, indPar)_indPar(k) holds the symmetric, positive definite property.
