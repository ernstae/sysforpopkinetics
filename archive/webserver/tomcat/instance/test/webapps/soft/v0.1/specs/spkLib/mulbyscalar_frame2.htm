Description
Performs element-wise matrix multiplication.

Returned Value
The void versions return the results through the third argument. Otherwise, the function returns the resulting m by n matrix by value.

Arguments
A
is a m by n matrix.

b
is a double-precision scalar multiplier.

B
is a m by n matrix multiplier.

C
is a reference to a m by n matrix. When this argument is given, the result is placed at the memory location pointed by the reference.

Example
If you compile, link and run the following program,
  #include "mulByScalar.h"

  void main(){
     DoubleMatrix A( 3, 4 );
     double scalar = 2.0;
     DoubleMatrix C;

    double *a = A.data();

    // Set both A to a matrix
     //     [ 1  4  7 ]
     //    [ 2  5  8 ]
     //    [ 3  6  9 ]

    for( int i=0; i<12; i++ ){
          a[i] = i+1;
     }
     
     // Computer C = A * scalar;
     C = mulByScalar( A, scalar );

    // Display C
     C.print();

  }
 a following matrix is displayed.

  [ 2   8  14 ]
  [ 4  10  16 ]
  [ 9  12  18 ]
