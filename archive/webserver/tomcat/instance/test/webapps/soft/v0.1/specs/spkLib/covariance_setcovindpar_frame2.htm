Description
Sets the current value for the full individual parameter vector equal to indParNew.

If this covariance matrix only depends on a subset of the elements in the full individual parameter vector, then this function only sets the values for that subset.

Note
This function should never be called by the Covariance member functions, neither directly nor indirectly. The reason for this is that the Covariance member functions must not modify the state of the Covariance objects.
