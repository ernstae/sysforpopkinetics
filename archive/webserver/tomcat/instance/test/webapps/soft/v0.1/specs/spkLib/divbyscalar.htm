Element-wise Division
Prototype: const DoubleMatrix divByScalar(const DoubleMatrix &A, double b)
void divByScalar(const DoubleMatrix &A, double b, DoubleMatrix &C)
const DoubleMatrix divByScalar(const DoubleMatrix &A, const DoubleMatrix B)
void divByScalar(const DoubleMatrix &A, const DoubleMatrix B, DoubleMatrix& C)
See also: addition , subtraction , matrix multiply , matrix back division , element-wise multiply .

Description
This function performs element-wise matrix division. If the second argument is a scalar, all the elements of A is divided by b. If it is a matrix with the same dimensions as of A, each element of A is divided by the corresponding element of B. The result is returned by value or is set to the memory location referred by C if given.

Arguments
A
is a m by n matrix numerator.

b
is a scalar denominator. b must be non-zero.

B
is a m by n matrix denominator.

C
is a m by n matrix that will contain the result if given. If the dimensions does not match, the program terminates.

Revisit
Division by zero exception should be handled properly.

Example
If you compile, link and run the following program,
  #include "divByScalar.h"

  void main(){
     DoubleMatrix A( 3, 4 );
     double scalar = 2.0;
     DoubleMatrix C;

    double *a = A.data();

    // Set both A to a matrix
     //     [ 1  4  7 ]
     //    [ 2  5  8 ]
     //    [ 3  6  9 ]

    for( int i=0; i<12; i++ ){
          a[i] = i+1;
     }
     
     // Computer C = A / scalar;
     C = divByScalar( A, scalar );

    // Display C
     C.print();

  }
 a following matrix is displayed.

  [ 0.5  2.0  3.5 ]
  [ 1.0  2.5  4.0 ]
  [ 1.5  3.0  4.5 ]
