Description
The copy constructor.

Since Covariance is an abstract class, the direct use of this constructor is only allowed for its subclasses.

The copy constructor is provided to make the Covariance class more standard even though SPK itself does not currently require Covariance objects to be copyable.

Arguments
right
The Covariance to be copied.
