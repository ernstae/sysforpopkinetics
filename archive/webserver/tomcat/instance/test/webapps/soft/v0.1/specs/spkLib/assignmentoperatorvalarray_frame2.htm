Description
This overloaded assignment operator assigns the element values and the size of an valarray object, array, to an column DoubleMatrix object if the size of array is not zero. If the size of array is zero, the operator assigns the left side DoubleMatrix object to a zero size DoubleMatrix object.

If there is no enough resource available in the heap, it throws SpkException of which the last SpkError is set to SPK_INSUFFICIENT_MEM_ERR.

UnitTest Plans
Check if a finite size column matrix is successfully assigned from a finite size valarray object. Check if a zero size matrix is successfully assigned from a zero size valarray object.
