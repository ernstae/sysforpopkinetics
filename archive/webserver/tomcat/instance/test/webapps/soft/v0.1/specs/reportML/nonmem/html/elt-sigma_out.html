Element <sigma_out>

A minimal representation of NONMEM Omega matrix when

optimization completes (ie. converges) successfully. If fails, this entry contains the value from the last successfully completed iteration. For full (ie. "block") matrix, sigma_out lists the lower triangle elements in the row-major order. For diagonal, sigma_out lists only the diagonal elements from the upper left corner. For block diagonal, each sigma_out block is listed individually using full or diagonal specifications as appropriate. Given a 3 by 3 matrix, A: 

/                   \
|  a11   a12   a13  |
|  a21   a22   a23  |
|  a31   a32   a33  |
\                   /
If A were full (ie. block) symmetric, sigma_out specifies as follows: 
    <sigma_out>
       <value>a11</value>
       <value>a21</value><value>a22</value>
       <value>a31</value><value>a32</value><value>a33</value>
    </sigma_out>
If A were diagonal, sigma_out specifies as follows: 
    <sigma_out>
       <value>a11</value>
                         <value>a22</value>
                                           <value>a33</value>
    </sigma_out>
If A were block diagonal symmetric with a 2x2 full block and a 1x1 diagonal block, sigma_out specifies as two blocks as follows: 
    <sigma_out>
       <value>a11</value>
       <value>a21</value><value>a22</value>
    </sigma_out>
    <sigma_out>
       <value>a33</value>
    </sigma_out>

Attributes

dimension REQUIRED

The dimension of the Simga matrix.

struct REQUIRED

The structure of the Sigma matrix: "block" or "diagonal".
