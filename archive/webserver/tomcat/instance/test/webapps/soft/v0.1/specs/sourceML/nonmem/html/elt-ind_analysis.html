Element <ind_analysis>

Request for the individual level analysis. The model definition is assumed to be for a single individual.

Attributes

sig_digits IMPLIED

The number of significant digits required in the final parameter estimate. The value must be a positive integer less than 9. The default value is 3.

is_restart IMPLIED

Request to start from the last state of the parent job's optimization process; "yes" or "no", default to "no".

mitr IMPLIED

Maximum iterations attemped before the optimizer gives in. Required when "is_estimation" is set to "yes".

abort IMPLIED

"no" is default. REVISIT Sachiko 05/26/04

is_estimation REQUIRED

"yes" for the parameter estimation to be performed. When set to "no", the parameter estimation will be skipped. This value should be "no" if monte_carlo is requested.
