Element <presentation_data>

The values that

appear on the left hand side of the equations in the model definition and the following extra items, both evaluated at each sampling point:

  • Observation record identifier (ID)
  • Prediction (PRED)
  • Residual (RES) (= DV - RRED)
  • Weighted residual (WRES) if population analysis were requested.
  • Simulated data (ie. dependent variable) if data simulation were requested (SIMDV).
with THETA set to:
  • initial values if no parameter estimation is requested.
  • final values if parameter estimation is requested.
The data set is considered as a table/matrix here and the first row in the table would contain the labels that describe the nature of the values that follow (column-wise).

Attributes

columns REQUIRED

The number of colums in the table, >=0.

rows REQUIRED

The number of rows in the table, >=0.

Content

(data_labels | row*)
