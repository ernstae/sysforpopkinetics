Element <omega>

A minimal representation of NONMEM Omega (symmetric positive definite) matrix. For full (ie. "block") matrix, omega lists the lower triangle elements in the row-major order. For diagonal, omega lists only the diagonal elements from the upper left corner. For block diagonal, each omega block is listed individually using full or diagonal specifications as appropriate.

Given a 3 by 3 matrix, A: 

/                   \
|  a11   a12   a13  |
|  a21   a22   a23  |
|  a31   a32   a33  |
\                   /
If A were full (ie. block) symmetric, omega specifies as follows: 
    <omega>
       <value>a11</value>
       <value>a21</value><value>a22</value>
       <value>a31</value><value>a32</value><value>a33</value>
    </omega>
If A were diagonal, omega specifies as follows: 
    <omega>
       <value>a11</value>
                         <value>a22</value>
                                           <value>a33</value>
    </omega>
If A were block diagonal symmetric with a 2x2 full block and a 1x1 diagonal block, omega specifies as two blocks as follows: 
    <omega>
       <value>a11</value>
       <value>a21</value><value>a22</value>
    </omega>
    <omega>
       <value>a33</value>
    </omega>

Attributes

struct REQUIRED

Structure of Omega matrix, either "block" or "diagonal". "block" suggests that any elements in the symmetric matrix could be non-zero. "diagonal" indicates all zeros except for the diagonal elements.

dimension REQUIRED

Dimension of square Omega matrix. For example, for a 3 by 3, say "3".
