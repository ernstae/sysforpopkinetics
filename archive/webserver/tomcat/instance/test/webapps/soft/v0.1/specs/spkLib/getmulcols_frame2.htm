Description
Returns a separate matrix that contains only these columns of a m by n matrix A that are selected by a selection row vector S. A given column in A is selected if the corresponding value in S is true (positive). The number of columns in the result is the number of true values in S. If A is empty (i.e. if m=0 or n=0), or if all values in S are false (zero or negative), then the result is an empty matrix (m=0 and n=0).

The order of resulting matrix preserves the original matrix's order.

The number of columns in A and S must match, and S must have one row. Otherwise, the program terminates.

Arguments

&A
is a m by n matrix whose rows are to be selected. The collection of these selected columns is returned as a matrix.

&S
is a 1 by n row vector that contains true (> 0.0) and false (<= 0.0) values. If s(j) = true, then the j-th column of A is said to be chosen and the column will be found in the resulting matrix. If s(j) = false, then the j-th row of A will be ignored.

Example
If you compile, link, and run the following program:

    #include <iostream>
    #include "getMulCols.h"
    #include "DoubleMatrix.h"

    void main(){
        using namespace std;

        int m = 3;
        int n = 3; 
        int i,j;
        DoubleMatrix A(m,n);
        DoubleMatrix S(1,n);
        DoubleMatrix C;
        double *pdA = A.data();
        double *pdS = S.data();

        // Set A to a matrix:
        //  [ 0  3  6 ]
        //  [ 1  4  7 ]
        //  [ 2  5  8 ]
        //
        // Set S to a vector:
        //  [ true, false, true] = [ 1, 0, 1 ]
        //
        for( j=0; j<n; j++ ){
            for( i=0; i<m; i++ ){
                pdA[i+j*m] = i+j*m;
            }
            pdS[j] = (double)(!(j % 2));
        }

        C = getMulCols(A,S);

        cout << "A = " << endl;
        A.print();

        cout << "\nS = " << endl;
        S.print();

        // Result =
        //  [0, 6]
        //  [1, 7]
        //  [2, 8]
        cout << "\nResult = " << endl;
        C.print();
    }

 then it will display the following when it is run:
    A =
    [0, 3, 6]
    [1, 4, 7]
    [2, 5, 8]

    S =
    [1, 0, 1]

    Result =
    [0, 6]
    [1, 7]
    [2, 8]
