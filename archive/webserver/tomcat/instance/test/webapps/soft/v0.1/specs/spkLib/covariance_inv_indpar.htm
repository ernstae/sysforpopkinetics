Covariance::inv_indPar & (pure virtual) doInv_indPar
Prototype: public: bool covInv_indPar( DoubleMatrix& \ret\ ) const;
private: virtual bool doCovInv_indPar( DoubleMatrix& \ret\ ) const = 0;

Description
The public interface void covInv_indPar( DoubleMatrix& \ret\ ) const evaluates the user defined virtual doCovInv_indPar( DoubleMatrix& \ret\ ) const at the current evaluation points.

Arguments
ret 
is n by nPopPar matrix resulted from evaluating doCovInv_indPar(), where n is the product of #rows and #cols of a matrix resulted from evaluating doCovInv() and nPopPar is the size of the individual parameter vector. j-th column of matrix, ret, contains the inverse of the derivative of the covariance with respect to j-th element of the individual parameter vector, in the column-major order. Each column, or the derivative with respect to j-th, must be a symmetric positive definite matrix.
