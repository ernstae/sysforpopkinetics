Description
Objective enumerates symbols for different objectives:

MODIFIED_LAPLACE
for the Modified Laplace objective,

EXPECTED_HESSIAN
for the Modified Expected Hessian (also known as FOCE) objective, and

FIRST_ORDER
for the Modified First Order objective.

Reference
B. M. Bell, Approximating The Marginal Likelihood Estimate For Models With Random Parameters, Applied Mathematics and Computation, Accepted 1999.

Example
If you compile and link the following program,
    #include <iostream>
    #include "Objective.h"

    int main()
    {
      using namespace std;

      Objective ob = MODIFIED_LAPLACE;

      switch(ob)
      {
        case MODIFIED_LAPLACE:
          cout << "modified laplace" << endl;
          break;
        case EXPECTED_HESSIAN:
          cout << "expected hessian" << endl;
          break;
        case FIRST_ORDER:
          cout << "first order" << endl;
          break;
        default:
          break;
      }
      return 0;
    }
