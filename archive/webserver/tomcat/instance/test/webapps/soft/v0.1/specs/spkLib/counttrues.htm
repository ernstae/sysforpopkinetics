Count Trues
Prototype: int countTrues(const DoubleMatrix& A)

Copyright (C) 2002, University of Washington, Resource Facility for Population Kinetics. All Rights Reserved.

Description
Returns the number of trues in a matrix. A value greater than 0.0 is considered true.

Arguments

A
is a m by n matrix containing true (>0) and/or false (=0) values, where m and n are greater than or equal to zero.

Example
If you compile, link, and run the following program:

    #include <iostream>
    #include "DoubleMatrix.h"

    void main()
    {
        using namespace std;

        int m = 3;
        int n = 2;

        //
        // Set a matrix that contains 2 trues.
        //
        DoubleMatrix A(m,n);
        double* pA = A.data();
        pA[0] = 0;
        pA[1] = 1;
        pA[2] = 0;
        pA[3] = 0;
        pA[4] = 0;
        pA[5] = 1;

        cout << "#of trues in A: " << countTrues(A) << endl;
    }
 then it will display the following when it is run:

    #of trues in A: 2
