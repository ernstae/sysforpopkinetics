Element <presentation_data>

This element is composed of two table data.

Each table data is given by a single string. The first table summerizes the invariants for all the individuals. The second table summerizes all the variants for individuals.

Invariants

The indviduals invariants section contains ETA and its residuals. Ixxx, Cxxx and Pxxx appear only when the population anlaysis has been performed.

ETA
Random parameters.
ETARES
Individual data residuals.
WETARES
Individual data weighted residuals. The above devided by the covariance of individual's data R(b) as the weight.
IETARES
Individual parameter residuals.
IWETARES
Weighted individual parameter residuals. The above devided by the covariance of individual parameters D(alp) as the weight.
CETARES
Conditional parameter residuals.
CWETARES
Weighted conditional parameter residuals.
PETARES
Individual's residuals in the context of population.
PWETARES
Weighted population residuals. The above devided by the the covariance of an individual's data as the weight.

Variants

The individuals variants section containts three classes of data:

  • The exact original data set or the original data with DV replaced by a newly simulated data set when simulation was requested. Also, the ID data item may be present if no ID value was given in the original set.
  • The quantities computed and stored in the variables that appear on the left hand side of the equations in the model definition,
  • The system defined variables that vary depending upon which canned-model you are using. The following items are always added:
    MDV
    Missing Dependent Variable if the data set did not have this data item in the first place.
    PRED
    Prediction.
    RES
    Residual for the prediction, where RES = DV - PRED.
    ORGDV
    The original DV value when a new data set is simulated. The new data set populates DV.
with THETA set to:
  • initial values if no parameter estimation is requested.
  • final values or values from the last successful & complete iteration if parameter estimation is requested.

NOTE: If ID was not given for an individual data set, the field would be populated with a value "1". For a population data set, sequencial numbers (beginning with 1) would be assigned.
