Element <monte_carlo>

If this element is present, an approximation for the integral and an estimate of its standard deviation will be computed.

Attributes

seed IMPLIED

Positive integer used as a seed to generate random numbers. The default value is 123.

method IMPLIED

Specifies an integration method. Four values are available: "plain", "analytic", "grid", "miser" and "vegas". "plain" (default) requests an approximation based on uniform sampling. "analytic" is only valid for LinearModel (consult Brad) and requests to output the analytic integral. "grid" requests a numerical approximation using a uniform grid. For "miser" requests to use the miser algorithm for approximating the integral described in Section 7.8 of Numerical Recipes (2nd Ed.). "vegas", based on importance sampling, is described at http://www.gnu.org/software/gsl/manual/gsl-ref_23.html#SEC375. For any of these method, an estimate of the standard deviation will be computed as well.
