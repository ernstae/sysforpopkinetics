Description
Return the transpose of a matrix A.

Return Value
Given a m by n matrix A, transpose(A) returns a n by m matrix B, such that (b(i,j) in B) = (a(j,i) in A), where m > i > 0 and n > j > 0.

Arguments
A
contains the value of a m by n matrix, where m and n are greater than zero.

Example
If you compile, link, and run the following program,

#include "DoubleMatrix.h"
#include "transpose.h"

void main(){
  
  DoubleMatrix A(3, 2);
  double *pA = A.data();
  int i;

  // Set A to the matrix
  // [ 1 4 ]
  // [ 2 5 ]
  // [ 3 6 ]
  for(i=0; i<6; i++)
    pA[i] = i+1;

  // Compute
  // [1 2 3] = [1 4]^T
  // [4 5 6]   [2 5]
  //           [3 6]
  DoubleMatrix B = transpose(A));
  B.print();
}
 the matrix
   [1 2 3]
   [4 5 6]
will be printed.
