Description
This constructor should be used to construct an empty matrix as a static matrix object. It keeps track of the number of the objects. If other constructors are used to create a static matrix object, it will appear as a memory leak.

identity specifies the filename in which the static object, A, appears. You can use an ANSI macro, __FILE__ to automatically obtain the information.
