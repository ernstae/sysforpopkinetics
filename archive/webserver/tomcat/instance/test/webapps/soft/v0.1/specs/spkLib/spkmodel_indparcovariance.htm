Description
getIndParCovariance() returns a constant reference to an object representing the individual parameters covariance (ie. doIndParVariance() ).

IndParCovariance is a subclass of Covariance class. See Covariance classes for more details.

Reference
B. M. Bell, Approximating The Marginal Likelihood Estimate For Models With Random Parameters, Applied Mathematics and Computation, Accepted 1999.
