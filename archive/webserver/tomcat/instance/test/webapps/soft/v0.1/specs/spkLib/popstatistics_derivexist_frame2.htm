Description
This function computes the covariance matrix, standard error vector, correlation matrix, coefficient of variation vector, and confidence interval vector for population parameter estimates. It allows parameter elements that are not active to be specified and removed from the statistics computations.

This function allows the covariance matrix of the population parameter estimates to be calculated using one of the following formulations:          

                                        -1     -1
    formulation "RSR":  cov[ alp ]  =  R   S  R   ;

                                        -1
    formulation "R":    cov[ alp ]  =  R   ;

                                        -1
    formulation "S":    cov[ alp ]  =  S   .

These formulations are discussed in Section (D.2.5) of the NONMEM Users Guide Part II and in Sections (A.2.1) and (A.2.2) in Carroll, Ruppert, and Stefanski (1998).

The approximation made for the information matrix is

                                          T
     R = ( LTilde_alp_alp + LTilde_alp_alp  ) / 2  ,

where LTilde_alp_alp is an approximation for the second order derivatives of the population objective function with respect to the population parameter alp. The matrix R is defined in this way to insure that it is symmetric even for cases where the approximation LTilde_alp_alp is not. Note that this R is different than R(b), which is the model for the covariance of an individual's data and is part of SpkModel .

The cross-product gradient matrix is defined as

                                    T
     S = Sum{ [(LambdaTilde_i )_alp] [(LambdaTilde_i)_alp] }  ,
          i                          

where (LambdaTilde_i)_alp is the derivative of the i-th individual's objective function with respect to the population parameter alp.

The standard error vector is calculated by taking the square roots of the diagonal elements of the covariance matrix. The correlation matrix is calculated by dividing each element of the covariance matrix by the standard errors that correspond to its row and column. The coefficients of variation are calculated as
   
    CV    =  SE    / | alp    | * 100   ,
     (i)       (i)        (i)

where CV is the coefficient of variation and SE is the standard error. The 95% confidence intervals are calculated as
   
    ( alp    -  t               * SE    ,  alp    +  t               * SE    ) ,
         (i)     0.025, degFree     (i)       (i)     0.025, degFree     (i)

where
   
    t
     0.025, degFree

is the critical value for the t distribution with degFree = nY - nAlp number of degrees of freedom for which the area under the t curve is 1 - 0.025 and nAlp is the number of population parameters.

Reference
Beal, S. L. and Sheiner, L. B. (1988) NONMEM Users Guide - Part II, University of California, San Francisco.

Carroll, R. J., Ruppert, D., and Stefanski, L. A. (1998) Measurement Error in Nonlinear Models, Chapman & Hall/CRC, Boca Raton, Florida.

Return Value
Upon a successful completion, the function sets the given output value place holders to point to the result value.

If an error is detected or failure occurs during the evaluation, a SpkException object is thrown. The state at which an exception is thrown is defined in Exception Handling Policy .

Arguments
mask
mask is a vector of boolean values of length equal to the parameter vector, popPar. mask[i] tells as to whether popPar[i] is active or not. If mask[i] is false, the i-th element of the parameter vector are treated as if it does not exist and further statistics computations are performed based upon the assumption.

measurementsAll
The SPK_VA::valarray<double> measurementsAll contains the vector y, which specifies the data for all the individuals. The vector y has
    N(1) + N(2) + ... + N(M)
elements where N(i) is the number of measurements for i-th individual and M is the total number of individuals in the population. The data vector corresponding to the first individual is                                          
    y_1 = [ y(1) , y(2) , ... , y(N(1)) ]
Elements y(N(1) + 1) through y(N(1) + N(2)) correspond to the second individual and so on. (Note that y_1 refers to the first subvector of y while y(1) refers to the first element of the valarray y.)

popPar
The SPK_VA::valarray<double> popPar contains the vector alp, which specifies the estimates of the population parameters. The returned covariance matrix popParCovOut will be evaluated at these values. The popPar should be obtained by calling the SPK function fitPopulation .

popObj_popPar_popPar 
The SPK_VA::valarray<double> popObj_popPar_popPar contains the matrix LTilde_alp_alp, in column major order, which specifies an approximation for the second derivative of the population objective function with respect to population parameter evaluated at popPar. Note that the size of popObj_popPar_popPar should be equal to the square of the length of the population parameter vector alp. The popObj_popPar_popPar should be obtained by calling the SPK function fitPopulation .

indObj_popParAll
The SPK_VA::valarray<double> indObj_popParAll contains the matrix LamdbdaTilde_alpAll, in column major order, each column of which specifies the transpose of one of the derivatives, (LambdaTilde_i)_alp. The size of indObj_popParAll is equal to the product of the length of the population parameter vector alp and the number of individuals in the population M. The i-th column of LamdbdaTilde_alpAll contains the transpose of the row vector (LambdaTilde_i)_alp, which specifies the derivative of the i-th individual's objective function with respect to the population parameter alp. Note that the row dimension of LamdbdaTilde_alpAll is equal to the length of the population parameter vector alp, and its column dimension is equal to the number of individuals in the population, M, The indObj_popParAll should be obtained by calling the SPK function lTilde .

formulation
The int formulation specifies which formulation of the covariance of the population parameter estimates is selected. See Description section for details. Only formulation "R" is available for FIRST_ORDER objective.

popParCovOut 
If popParCovOut is not NULL, then the SPK_VA::valarray<double> object pointed to by popParCovOut must be declared in the function that calls this function, and its size must be equal to the square of the length of the population parameter vector alp. If popParCovOut is not NULL and this function completed successfully, then the SPK_VA::valarray<double> object pointed to by popParCovOut will contain the covariance matrix of the population parameter estimates, in column major order, that is evaluated at alp. Otherwise, this function will not attempt to change the contents of the SPK_VA::valarray<double> object pointed to by popParCovOut. The (i,j)-the element of the covariance matrix will be replaced by NaN if mask[i] or mask[j] is false.

popParSEOut 
If popParSEOut is not NULL, then the SPK_VA::valarray<double> object pointed to by popParSEOut must be declared in the function that calls this function, and its size must be equal to the length of the population parameter vector alp. If popParSEOut is not NULL and this function completed successfully, then the SPK_VA::valarray<double> object pointed to by popParSEOut will contain the standard error vector of the population parameter estimates, in column major order, that is evaluated at alp. Otherwise, this function will not attempt to change the contents of the SPK_VA::valarray<double> object pointed to by popParSEOut. The i-th element of the standard error vector will be replaced by NaN if mask[i] is false.

popParCorOut 
If popParCorOut is not NULL, then the SPK_VA::valarray<double> object pointed to by popParCorOut must be declared in the function that calls this function, and its size must be equal to the square of the length of the population parameter vector alp. If popParCorOut is not NULL and this function completed successfully, then the SPK_VA::valarray<double> object pointed to by popParCorOut will contain the correlation matrix of the population parameter estimates, in column major order, that is evaluated at alp. Otherwise, this function will not attempt to change the contents of the SPK_VA::valarray<double> object pointed to by popParCorOut. The (i, j)-th element of the correlation matrix will be replaced by NaN if mask[i] or mask[j] is false.

popParCVOut 
If popParCVOut is not NULL, then the SPK_VA::valarray<double> object pointed to by popParCVOut must be declared in the function that calls this function, and its size must be equal to the length of the population parameter vector alp. If popParCVOut is not NULL and this function completed successfully, then the SPK_VA::valarray<double> object pointed to by popParCVOut will contain the standard error vector of the population parameter estimates, in column major order, that is evaluated at popPar. Otherwise, this function will not attempt to change the contents of the SPK_VA::valarray<double> object pointed to by popParCVOut. The i-th element of the coefficient vector will be replaced by NaN if mask[i] is false.

popParCIOut 
If popParCIOut is not NULL, then the SPK_VA::valarray<double> object pointed to by popParCIOut must be declared in the function that calls this function, and its size must be equal to the two times of the length of the population parameter vector alp. If popParCIOut is not NULL and this function completed successfully, then the SPK_VA::valarray<double> object pointed to by popParCIOut will contain the 95% confidence interval values of the population parameter estimates, in column major order, that is evaluated at popPar. There are two columns in the object. The first column contains the lower limit, and the second column contains the upper limit of the confidence interval of the population parameter estimates. Otherwise, this function will not attempt to change the contents of the SPK_VA::valarray<double> object pointed to by popParCIOut. Note that in the calculation of the confidence interval, if the degree of freedom (total number of data - number of population parameters) is greater than 120, it is treated as infinite. The (i,1) and (i,2) elements of the confidence interval matrix will be replaced by NaN if mask[i] is false.
