Description
Required for both population and individual analysis.

SpkModel<Scalar>::doDataVariance() calculates the variance of a particular individual's data for a currently selected individual (see selectIndividual ) at a currently set individual parameter (see setIndPar ) and population parameter (see setPopPar ).

The evaluation result of doDataVariance() shall be placed in ret and the function returns nothing.

doDataVariance() shall be only called via its corresponding public interface, dataVariance(). The public interface shall be called only by the SPK library components unless when it is being unit-tested, or SPK cannot guarantee any of the computational results.

The public interface, dataVariance() catches any exception thrown by the private interface, doDataVariance(), implemented by the user. If the exception is an instance of SpkException , an additional SpkError object will be appended to the list of errors maintained within the object. The new SpkError object will have error code set to SpkError::SPK_MODEL_DATA_VARIANCE_ERR.

Reference
B. M. Bell, Approximating The Marginal Likelihood Estimate For Models With Random Parameters, Applied Mathematics and Computation, Accepted 1999.

Arguments
ret
shall, upon a successful completion of evaluation, contain a resulting vector of size nYi * nYi, where nYi is the number of currently selected individual's data. The vector contains a column-ordered matrix of dimensions, nYi by nYi, that is symmetric and positive definite.
