Description
Required for population analysis.

doSetPopPar( SPK_VA::valarray<Scalar> popPar ) sets popPar as the current population parameter value and guarantees that subsequent attempts for evaluating the mathematical models expressed by SpkModel virtual member functions are evaluated at the point.

doSetPopPar() shall be only called via its corresponding public interface, setPopPar(). The public interface shall be called only by the SPK library components unless when it is being unit-tested, or SPK cannot guarantee any of the computational results.

The public interface, setPopPar() catches any exception thrown by the private interface, doSetPopPar(), implemented by the user. If the exception is an instance of SpkException , an additional SpkError object will be appended to the list of errors maintained within the object. The new SpkError object will have error code set to SpkError::SPK_MODEL_SET_POP_ERR.

Arguments
popPar
is a population parameter vector of size nPopPar, where nPopPar > 0.
