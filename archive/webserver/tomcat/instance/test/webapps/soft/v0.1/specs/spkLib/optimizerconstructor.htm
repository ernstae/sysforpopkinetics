Description
Constructor for the class.

The last three arguments for this constructor are optional and only need to be provided when restart information for this Optimizer object needs to saved to or retrieved from a restart file.

Arguments
epsilonIn
This real number is used to specify the convergence criteria for the optimizer. It must be greater than 0.0.

nMaxIterIn
This integer must be greater than or equal to zero. It specifies the maximum number of iterations to attempt before giving up on convergence.

levelIn
This integer scalar specifies the amount of tracing. Larger values of levelIn entail more tracing.

restartFileNameIn
This string specifies the name of the XML format restart file.

readRestartInfoIn
This bool specifies whether or not restart information should be read from the restart file.

writeRestartInfoIn
This bool specifies whether or not restart information should be written to the restart file.

Other data members

nIterCompleted
This integer scalar holds the number of iteration that have been completed in the optimizer. The initial value of 0 is set at the construction time.

isTooManyIter
This flag indicates whether the too-many-iteration failure has occurred. It is set to false at the construction time.

saveStateAtEndOfOpt
This flag indicates if the state information required for a warm start should be saved at the end of the optimization process. It is set to false at the construction time.

throwExcepIfMaxIter
This flag indicates if the optimizer should throw an exception when the maximum number of iterations is exhausted. It is set to true at the construction time.

isWarmStartPossible
This flag indicates whether it is possible to perform a warm start using the current optimizer state information. It is set to false at the construction time.

isWarmStart
This flag indicates whether the optimization should run a warm start. It is set to false at the construction time.

didOptFinishOk
This flag indicates whether the optimizer completed without an error occurring during the previous time it was called. It is set to false at the construction time.

isBeginOfIterStateInfo
This flag indicates whether the current optimizer state information is from the beginning of the last iteration during the previous time the optimizer was called. It is set to false at the construction time.

stateInfo
This stateInfo structure contains the optimizer state information required to perform a warm start. Each of its elements is described separately below.

stateInfo.n
The element n specifies the number of components in the element vector x.

stateInfo.b
The element b specifies the number of Bfgs updates that have been made to the Hessian approximation h.

stateInfo.r
The element r contains the current trust region radius (as an infinity norm bound on the step size).

stateInfo.f
The element f contains the value for f(x) at the point x.

stateInfo.x
The element x is a vector of length n. It specifies the point at which the objective function, its gradient, and its Hessian were evaluated.

stateInfo.g
The element g is a vector of length n. It contains the gradient of f(x) at the point x.

stateInfo.h
The element h is a vector of length n * n. It contains an approximation for the Hessian of f(x) at the point x.

stateInfo.m
The element m specifies the total number of objective function parameters, i.e., the number of free parameters plus the number of parameters that are constrained by both their lower and upper bounds.

stateInfo.low
The element low is a vector of length m. It specifies the lower bounds for all of the objective function parameters in their original coordinates.

stateInfo.up
The element up is a vector of length m. It specifies the upper bounds for all of the objective function parameters in their original coordinates.

stateInfo.pos
The element pos is a vector of length n. It specifies the positions, i.e., indices, of the free objective function parameters in the full objective function parameter.

stateInfo.acceptStepCount
The element acceptStepCount specifies the number of consecutive iterations that acceptable step values have been calculated.
