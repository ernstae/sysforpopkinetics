Description
Generates the mean value of each row in a given matrix V and stores this value in a vector.

The formula for the mean of a single row of a matrix is:

      M
  1  ---
  -  \   V[i,j]                              
  M  /__            
     j=1

  where M is the number of columns in V and V[i,j] is the ith row 
  and jth column of V.


Return Value
The i-th element of the resulting vector corresponds to the mean of i-th row of matrix V.

Arguments
V
V stores a Q by M matrix in column major order, where Q and M > 0.
Q
Q is the number of rows in the matrix V.

Example
If you compile, link, and run the following program,
     
    #include <iostream>
    #include <spk/SpkValarray.h>
    #include <spk/calcMean.h>

     void main()
     {
          using namespace std;

          valarray V(3 * 2), Mean(3);

          // Setting V to a matrix:
          //    [ 1  4 ]
          //    [ 2  5 ]
          //    [ 3  6 ]

          for (int i = 0; i < V.size(); i++ )
          {
               pV[i] = i+1;
          }

          Mean = calcMean(V, 3);

          cout << "Mean of V: " << Mean << endl;
    }

The output:
Mean of V: { 2.5, 3.5, 4.5 }
will be printed.
