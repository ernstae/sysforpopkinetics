Description
nc() returns the number of columns in the matrix as an integer value.

UnitTest Plans
Check things work for a 0x0, 0x1, 0xn, 1x0, 1xn, nxn matrix.

Example
If you compile, link and run the following program

#include <iostream>
#include "DoubleMatrix.h"
using namespace std;

void main(){

     DoubleMatrix A(3, 2);

     cout << A.nc() << endl;
}

 an integer value  
2 
will be printed.
