Description
Returns the norm of a (double) matrix. The norm is the square root of the sum of the squares of its elements.

Arguments

&A
m by n arbitrary matrix.

Example
If you compile, link, and run the following program:

    #include <iostream>
    #include "norm.h"

    void main()
    {
        using namespace std;

        DoubleMatrix dmatA(3,3);
        double *pdA = dmatA.data();

        // set the matrix to:
        // 
        // [0, 3, 2]
        // [1, 0, 3]
        // [2, 1, 0]
        //
        for( int i=0; i<3*3; i++ )
            pdA[i] = i%4;

        cout << "norm =  " << norm(dmatA) << endl;
    }

 then it will display the following when it is run:

    norm = 5.2915
