Description
This function evaluates the mean of all individual's data at a currently set population parameters of the original SpkModel class object and at zero value of individual parameters of the original SpkModel class object.

Arguments
f
shall, upon a successful completion of evaluation, contain a resulting vector of size nY, where nY is the number of all individual's data, in the following form.

          /  f1(alp, 0)  \
f(alp) =  |   :          |
          \  fn(alp, 0)  /

where f1,...,fn are the data means of individual 1,...,n of the original SpkModel class object, respectively. alp is the population parameter vector of the original SpkModel class object and n is the number of individuals of the original SpkModel class object.
