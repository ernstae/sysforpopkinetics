Description
Constructs a (size of array)/(nc) by nc matrix that contains the double values that are contained in the valarray object, array. The number of columns, nc, must be evenly divide the size of the valarray object, array. The order of the values copied to the matrix are arranged in column-major. If the size of array is zero, a zero size DoubleMatrix object is created.

If there is no enough resource available in the heap, it throws SpkException of which the last SpkError is set to SPK_INSUFFICIENT_MEM_ERR.

UnitTest Plans
Check if a finite size matrix is successfully created from a finite size valarray object. Check if a zero size matrix is successfully created from a zero size valarray object.
