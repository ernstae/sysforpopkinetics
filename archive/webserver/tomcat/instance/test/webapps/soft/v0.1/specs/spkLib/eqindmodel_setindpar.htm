Set Individual Parameter
Prototype:
void EqIndModel::doSetIndPar( const valarray<double>&  b ) 
Copyright (C) 2002, University of Washington, Resource Facility for Population Kinetics. All Rights Reserved.

Description
This function provides a means of setting individual parameters for the EqIndModel class object. The individual parameters of the EqIndModel class object are the population parameters of the original SpkModel class object.

Arguments
b
is an array of double values. The size of the array is the number of the population parameters of the original SpkModel class object.
