Description
This function computes the covariance matrix, standard error vector, correlation matrix, coefficient of variation vector, and confidence interval vector for individual parameter estimates. It allows parameter elements that are not active to be specified and removed from the statistics computations.

This function allows the covariance matrix of the individual parameter estimates to be calculated using one of the following formulations:

                                      -1     -1
    formulation "RSR":  cov[ b ]  =  R   S  R   ;

                                      -1     -1
    formulation "HSH":  cov[ b ]  =  H   S  H   ;

                                      -1
    formulation "R":    cov[ b ]  =  R   ;

                                      -1
    formulation "H":    cov[ b ]  =  H   ;

                                      -1
    formulation "S":    cov[ b ]  =  S   .

These formulations are discussed in Section (D.2.5) of the NONMEM Users Guide Part II and in Sections (A.2.1) and (A.2.2) in Carroll, Ruppert, and Stefanski (1998).

The first approximation that can be made for the information matrix is
                -                                     -
            1  |                                    T  | 
     R  =  --- | d  d  MapObj(b)  +  d  d  MapObj(b)   |  ,
            2  |  b  b                b  b             | 
                -                                     -
which is a symmetrized version of the Hessian of the individual objective function MapObj(b). The matrix R is defined in this way to insure that it is symmetric even for cases where the Hessian approximation is not.

The second approximation that can be made for the information matrix is
                -               -
               |                 | 
     H  =  E   | d  d  MapObj(b) | 
            y  |  b  b           | 
                -               -

            -1            T  -1                1        T        -1      -1
        =  D    +  d  f(b)  R  (b) d  f(b)  +  - d  R(b)  kron[ R  (b), R  (b) ] d  R(b)  ,
                    b               b          2  b                               b

which is the expected value with respect to the data y of the Hessian of the individual objective function. See Section (7.) of Bell (2001) for its derivation. Note that the R(b) on the right hand side of this equation is the model for the covariance of an individual's data, which is part of SpkModel and is different than the information matrix R. The term involving the covariance of the individual parameters D, which is also part of SpkModel , can be dropped from this calculation if the Map Bayesian objective function was not used to obtain the individual parameter estimates. See fitIndividual for details.

Currently, if either the covariance of the individual's data R(b) or the covariance of the individual parameters D is not diagonal, then only the R formulation may be used. If both are diagonal, however, the other two formulations may be used and the cross-product gradient matrix is defined as

            nY                  nY + nB
            ---                   ---
            \       T             \        T
     S  =   /    { s   s  }  +    /     { s   s  }  ,
            ---     i   i         ---      i   i
           i = 1               i = nY + 1

where nY is the number of data values for the individual and nB is the number of individual parameters. This is the empirical version of equation (A.2) from Carroll, Ruppert, and Stefanski (1998) where s_i is the likelihood score. The second sum can be dropped from this calculation if the Map Bayesian objective function was not used to obtain the individual parameter estimates. For i values less than or equal to nY,
                    -                                                                      -
           - 1     |                                           T     -1                     |
    s   =   --- d  | log[ 2 pi R     (b) ]  +  [y    - f   (b)]  R     (b) [y    - f   (b)] |  .
     i       2   b |            (i,i)            (i)    (i)       (i,i)      (i)    (i)     |
                    -                                                                      -

If the Map Bayesian objective function was used to obtain the individual parameter estimates, then there is one additional likelihood score for each of the individual parameters. For i values greater than nY,

                    -                                                   -
           - 1     |                                        -1        2  |
    s   =   --- d  | log[ 2 pi D            ]  +  D             b        |  .
     i       2   b |            (i-nY,i-nY)        (i-nY,i-nY)   (i-nY)  |
                    -                                                   -

The standard error vector is calculated by taking the square roots of the diagonal elements of the covariance matrix. The correlation matrix is calculated by dividing each element of the covariance matrix by the standard errors that correspond to its row and column. The coefficients of variation are calculated as

    CV    =  SE    / | b    | * 100   ,
     (i)       (i)      (i)

where CV is the coefficient of variation and SE is the standard error. The 95% confidence intervals are calculated as

    ( b    -  t               * SE    ,  b    +  t               * SE    ) ,
       (i)     0.025, degFree     (i)     (i)     0.025, degFree     (i)

where

    t
     0.025, degFree

is the critical value for the t distribution with degFree = nY - nB number of degrees of freedom for which the area under the t curve is 1 - 0.025.

Reference

Bell, B. M. (2001) Approximating the marginal likelihood estimate for models with random parameters, Applied Mathematics and Computation, 119, 57-75.

Carroll, R. J., Ruppert, D., and Stefanski, L. A. (1998) Measurement Error in Nonlinear Models, Chapman & Hall/CRC, Boca Raton, Florida.

Return Value
Upon a successful completion, the function sets the given output value place holders to point to the result values.

If an error is detected or failure occurs during the evaluation, a SpkException object is thrown. The state at which an exception is thrown is defined in Exception Handling Policy .

Arguments
model
This function expects model to be a function of b. Refer to Model Functions Depend on only b for details. If the Map Bayesian objective function was used to obtain the individual parameter estimates, then the function Model for the Variance of Individual Parameter must be defined for this model in order to calculate the covariance of the individual parameters D. In this case, the argument withD should be set equal to true.

measurements
The SPK_VA::valarray<double> measurements contains the array y, which specifies the measured data.

indPar
The SPK_VA::valarray<double> indPar contains the vector b, which specifies the estimates of the individual parameters. The returned values indParCovOut, indParSEOut, indParCorOut, indParCVOut and indParCIOut will be evaluated at these estimates. The values of indPar should be obtained by calling the SPK function fitIndividual .

indParMask
indParMask is a vector of boolean values of length equal to the parameter vector, indPar. indParMask[i] tells as to whether indPar[i] is active or not. If indParMask[i] is false, the i-th element of the parameter vector are treated as if it does not exist and further statistics computations are performed based upon the assumption.

indObj_indPar_indPar 
The SPK_VA::valarray<double> indObj_indPar_indPar contains the matrix MapObj_b_b, in column major order, which specifies an approximation for the second derivative of the individual objective function with respect to the individual parameter evaluated at indPar. Note that the size of indObj_indPar_indPar should be equal to the square of the length of the individual parameter vector b. The indObj_indPar_indPar should be obtained by calling the SPK function fitIndividual .

indParCovOut
If indParCovOut is not NULL, then the SPK_VA::valarray<double> object pointed to by indParCovOut must be declared in the function that calls this function, and its size must be equal to the square of the length of the individual parameter vector b. If indParCovOut is not NULL and this function completed successfully, then the SPK_VA::valarray<double> object pointed to by indParCovOut will contain the covariance matrix of the individual parameter estimates, in column major order, that is evaluated at indPar. Otherwise, this function will not attempt to change the contents of the SPK_VA::valarray<double> object pointed to by indParCovOut. The (i,j)-the element of the covariance matrix will be replaced by NaN if indParMask[i] or indParMask[j] is false.

indParSEOut
If indParSEOut is not NULL, then the SPK_VA::valarray<double> object pointed to by indParSEOut must be declared in the function that calls this function, and its size must be equal to the length of the individual parameter vector b. If indParSEOut is not NULL and this function completed successfully, then the SPK_VA::valarray<double> object pointed to by indParSEOut will contain the standard error vector of the individual parameter estimates, in column major order, that is evaluated at indPar. Otherwise, this function will not attempt to change the contents of the SPK_VA::valarray<double> object pointed to by indParSEOut. The i-th element of the standard error vector will be replaced by NaN if indParMask[i] is false.

indParCorOut
If indParCorOut is not NULL, then the SPK_VA::valarray<double> object pointed to by indParCorOut must be declared in the function that calls this function, and its size must be equal to the square of the length of the individual parameter vector b. If indParCorOut is not NULL and this function completed successfully, then the SPK_VA::valarray<double> object pointed to by indParCorOut will contain the correlation matrix of the individual parameter estimates, in column major order, that is evaluated at indPar. Otherwise, this function will not attempt to change the contents of the SPK_VA::valarray<double> object pointed to by indParCorOut. The (i, j)-th element of the correlation matrix will be replaced by NaN if indParMask[i] or indParMask[j] is false.

indParCVOut
If indParCVOut is not NULL, then the SPK_VA::valarray<double> object pointed to by indParCVOut must be declared in the function that calls this function, and its size must be equal to the length of the individual parameter vector b. If indParCVOut is not NULL and this function completed successfully, then the SPK_VA::valarray<double> object pointed to by indParCVOut will contain the standard error vector of the individual parameter estimates, in column major order, that is evaluated at indPar. Otherwise, this function will not attempt to change the contents of the SPK_VA::valarray<double> object pointed to by indParCVOut. The i-th element of the coefficient vector will be replaced by NaN if indParMask[i] is false.

indParCIOut
If indParCIOut is not NULL, then the SPK_VA::valarray<double> object pointed to by indParCIOut must be declared in the function that calls this function, and its size must be equal to the two times of the length of the individual parameter vector b. If indParCIOut is not NULL and this function completed successfully, then the SPK_VA::valarray<double> object pointed to by indParCIOut will contain the 95% confidence interval values of the individual parameter estimates, in column major order, that is evaluated at indPar. There are two columns in the object. The first column contains the lower limit, and the second column contains the upper limit of the confidence interval of the individual parameter estimates. Otherwise, this function will not attempt to change the contents of the SPK_VA::valarray<double> object pointed to by indParCIOut. Note that in the calculation of the confidence interval, if the degree of freedom (number of data - number of parameters) is greater than 120 it is treated as infinite. The (i,1) and (i,2) elements of the confidence interval matrix will be replaced by NaN if indParMask[i] is false.

withD
If this flag is set equal to false, then the terms involving the covariance of the individual parameters D are dropped from the calculations of the information matrix R and the cross-product gradient matrix S. This should only be done if the Map Bayesian objective function was not used to obtain the individual parameter estimates. See fitIndividual for details.

If this flag is set equal to true, then the function Model for the Variance of Individual Parameter must be defined for the input argument model in order to calculate D.
