Prototype istream& DoubleMatrix::operator>>(istream& is, DoubleMatrix& ob)

Copyright (C) 2002, University of Washington, Resource Facility for Population Kinetics. All Rights Reserved.

Description
The operator extracts a DoubleMatrix structure from is and puts it into ob. is is assumed to have the same exact format as a DoubleMatrix object for the matrix inserted to ostream by DoubleMatrix::operator<<. If is does not obey the format, the program may terminate. For a column-major 3 by 2 matrix, the following input is expected:
          2 by 3
          [ a11  a12 ]
          [ a21  a22 ]
          [ a31  a32 ]
 For an empty matrix, it should contain nothing except for the dimensions:
          0 by 0
          [ ]
