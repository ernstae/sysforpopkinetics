Description
Sets the matrix Chol equal to the lower triangular Cholesky factor of the current value of the covariance matrix Cov such that

                    T
  Cov  =  Chol  Chol   .

Note
(1.) This function is declared virtual and can be redefined in subclasses of this class.

(2.) This function should not make use of any cached values. Thus, if this function is redefined, it's new implementation does not need to call the cache update function.

Arguments
dmatChol 
On output this matrix will contain the lower triangular Cholesky factor of the current value for the covariance matrix Cov. Its dimensions may be different from what they were on input. When it fails to compute a Cholesky factorization, it throws an SpkException exception.
