Description
Writes to the restart file a subelement of the current element that contains a single "value" element, i.e., <current_element> ... <subelement> <value>########</value> </subelement> ... </current_element> where the sequence of number signs (#) represents the sequence of characters that constitute a single value.

This function assumes that the current element start tag has been written to the restart file before this function is called and that its end tag will be written after.

Arguments
ValType
The type of the value.

pSubelementTag
This argument specifies the name of the subelement of the current element that contains the single "value" element.

valueIn
This argument will be written as the contents of the single "value" element using the std::ostream insertion operator (<<), which converts it from type ValueType to a sequence of characters.
