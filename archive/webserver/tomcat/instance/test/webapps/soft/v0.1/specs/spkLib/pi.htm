Description
PI is a double-precision value defined as an approximation of pi.

Example
If you compile, link, and run the following program:

    #include <iostream>
    #include "PI.h"

    bool testPI()
    {
        using namespace std;
        cout << "PI = " << pi << endl;
        return true;
    }
