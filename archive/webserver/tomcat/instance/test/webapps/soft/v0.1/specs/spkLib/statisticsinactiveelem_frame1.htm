Computing Statistics of Parameter Estimates when Some Elements are not Active
void statistics( const SPK_VA::valarray<bool>  & mask,
           const SPK_VA::valarray<double> & x,       
           const SPK_VA::valarray<double> & xCov,    
           int                              degFree, 
           SPK_VA::valarray<double>       * seOut,            
           SPK_VA::valarray<double>       * corOut,  
           SPK_VA::valarray<double>       * cvOut,   
           SPK_VA::valarray<double>       * ciOut
                 )

Description
This function computes the standard error vector, correlation matrix, coefficient of variation vector, and confidence interval vector of x' based on a given covariance matrix of x. x' is a subset of x such that { x' | x AND mask } ("AND" is logical AND).

The standard error vector is calculated by taking the square roots of the diagonal elements of the covariance matrix. The correlation matrix is calculated by dividing each element of the covariance matrix by the standard errors that correspond to its row and column. The coefficients of variation are calculated as
   
    CV    =  SE    / | x    | * 100   ,
     (i)       (i)      (i)

where CV is the coefficient of variation and SE is the standard error. The 95% confidence intervals are calculated as
   
    ( x    -  t               * SE    ,  x    +  t               * SE    ) ,
       (i)     0.025, degFree     (i)     (i)     0.025, degFree     (i)

where
   
    t
     0.025, degFree

is the critical value for the t distribution with degFree number of degrees of freedom for which the area under the t curve is 1 - 0.025.

Return Value
Upon a successful completion, the function sets the given output value place holders to point to the result values.

If an error is detected or failure occurs during the evaluation, a SpkException object is thrown. The state at which an exception is thrown is defined in Exception Handling Policy .

Arguments
mask
is a vector of boolean values (ie. true/false) of length equal to the length of x. A value true in the i-th element of mask indicates that the i-th value of x has contributed to the parameter optimization.

x
is an n dimensional parameter vector.

xCov
is a covariance matrix of x.

degFree
is the degree of freedom used to compute the confidence interval. It is defined as degFree = m - n, where n is the length of parameter vector, in this case, x, and m is the number of fitted data points. The value must be greater than zero, although it will not be referenced at all if ciOut is set to NULL.

seOut
will point to an n dimensional vector containing the standard error for x if the pointer points to an n dimensional valarray. The elements of which corresponding mask value is false will be set to NAN. If it points to NULL, the corresponding computation will be skipped entirely. If it points to a valarray sized other than n, the resulting behavior is undetermined.

corOut
will point to an n * n dimensional vector containing the correlation matrix in the column major order if the pointer points to an n * n valarray. The elements depend on those x(i) of which corresponding mask is false will be set to NAN. If it points to NULL, the corresponding computation will be skipped entirely. If it points to a valarray sized other than n * n, the resulting behavior is undetermined.

cvOut
will point to an n dimensional vector containing the coefficient of variance if the pointer points to an n dimensional valarray. The elements of which corresponding mask value is false will be set to NAN. If it points to NULL, the corresponding computation will be skipped entirely. If it points to a valarray sized other than n, the resulting behavior is undetermined.

ciOut
will point to an n * 2 dimensional vector containing the 95% confidence interval if the pointer points to an n * 2 dimensional valarray. The elements of which corresponding mask value is false will be set to NAN. If it points to NULL, the corresponding computation will be skipped entirely. If it points to a valarray sized other than n * 2, the resulting behavior is undetermined.
