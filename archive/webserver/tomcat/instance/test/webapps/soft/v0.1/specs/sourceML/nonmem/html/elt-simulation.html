Element <simulation>

Request to simulate a data set.

If this element appears when the population analysis is chosen, a data set for n number of individuals will be generated, where n is the number of individuals in the population. If it is for the individual analysis, a data set for a single individual will be generated. The simulated data set replaces the original data set for parameter estimation if parameter estimation is requested.

Attributes

seed REQUIRED

Positive integer used as a seed to generate random numbers.

subproblems IMPLIED

Requests that the entire problem is to be repeated n times in succession (including all steps: simulation, estimation, statistics, table/scatterplot generation). Each subproblem includes the simulation step, but the random sources are simply continued from subproblem to subproblem. By default, n=1, there is only one subproblem.
