Element <comp_model>

Definition of a compartmental model. Required when a general model (ADVAN5-9) is used.

comp_model would usually follow pk block. pk is omitted if links are specified to define the compartmental model. The order in which the compartments appear is interpreted as the compartment number.

Attributes

nequilibriums IMPLIED

The number of equilibrium compartments, default to 0.

ncompartments IMPLIED

Total number of compartments other than the output compartment. The default is the number of compartments.

nparameters IMPLIED

The number of basic PK parameters. The default is the number of bsic PK parameters defined in the pk block.
