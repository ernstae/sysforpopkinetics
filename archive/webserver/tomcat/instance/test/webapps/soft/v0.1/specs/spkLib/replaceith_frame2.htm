Description
Replaces the ith row of target with the data in rowvec.

Arguments

target
is a m by n matrix, whose ith row will be replaced.

ith
indicates the row that is to be replaced. ith must be greater than or equal to 0 and less than m.

rowvec
is a n dimensional row vector of values that will replace the ith row of target.

Example
If you compile, link, and run the following program:

    #include <iostream>
    #include "DoubleMatrix.h"
    #include "replaceIth.h"

    void main()
    {
        const int m = 3;
        const int n = 2;

        //
        // initialize target to a m by n matrix filled with 0.0.
        DoubleMatrix target(m,n);
        target.fill(0.0);

        DoubleMatrix rowvec(1,n);
        target.fill(1.0);

        cout << "Before = " << endl;
        target.print();

        replaceIth(target, 1, rowvec);

        cout << "After = " << endl;
        target.print();
    }
 then it will display the following when it is run:

    Before = 
    [ 0 0 ]
    [ 0 0 ]
    [ 0 0 ]

    After =
    [ 0 0 ]
    [ 1 1 ]
    [ 0 0 ]
