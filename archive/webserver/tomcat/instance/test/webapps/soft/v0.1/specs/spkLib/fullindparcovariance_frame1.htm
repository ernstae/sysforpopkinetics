Description
This is a subclass of FullCovariance class which is derived from Covariance class as shown below:

                ------------------------
               |       Covariance       |    
                ------------------------
                       (abstract)
                /|                  /|           
                 |                    |
                 |                    |
     ------------------------   ------------------------ 
    |     FullCovariance     | |     DiagCovariance     |
     ------------------------  -------------------------
            (abstract)                 (abstract)
                /|                        /|         
                 |                          |
                 |                          |
    -------------------------   -------------------------
   |   FullIndParCovariance  | |   DiagIndParCovariance  | 
    -------------------------   -------------------------
            (concrete)                 (concrete)
                         
This class encapsulates math operations and state setting functions unique to the full covariance between individual parameters.
