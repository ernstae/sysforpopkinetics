Description
Copies information from right to *this and returns this, where *this is the Optimizer object on the left side of the = sign.

The *this object will be identical to the right, except for the freeXMLMemory flag, which is set equal to false, and except for the following pointers, which are all set equal to zero: the pointer to the objective function, pObjFunc; the pointer to the current XML element in the restart file, pCurrElement; and the pointer to the XML parser, pParser.
