Diacritical Marks Character Entities (%isodia;)

%isodia; — Diacritical Marks Character Entities

Synopsis

The %isodia; parameter entity includes the ISO character entities with the public identifier:

  ISO 8879:1986//ENTITIES Diacritical Marks//EN


Description

The following character entities are defined in this entity set:
