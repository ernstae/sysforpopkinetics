Greek Letters Character Entities (%isogrk1;)

%isogrk1; — Greek Letters Character Entities

Synopsis

The %isogrk1; parameter entity includes the ISO character entities with the public identifier:

  ISO 8879:1986//ENTITIES Greek Letters//EN
