NAME

regression_test -- run the production regression test

SYNOPSIS

regression_test

DESCRIPTION

This shell script runs regression_test.pl against the production regression test configuration file.

Like regression_test.pl, regression_test must be run on the aspkserver, and must be run with root privilege in root's environment. To insure that this happens, log into aspkserver as an ordinary user, then use su - (the minus sign is crucial) to gain root privilege.

OPTIONS AND ARGUMENTS

None.

EXIT STATUS

Returns the value 0 as exit status if all tests were successful; otherwise, a non-zero value is returned.

FILES

aspkserver:/usr/local/spk/ops/regression_test/config/regression_test.cml

SEE ALSO

The man page for regression_test.pl, which is on the web page where you obtained this manual page, or can be obtained by running the command regression_test.pl --man
