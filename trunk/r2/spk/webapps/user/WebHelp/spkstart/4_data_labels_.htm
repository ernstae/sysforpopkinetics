4. Data Labels.

In the Data Labels screen, enter names for the data columns. In a population analysis, the first column must contain the ID codes for the population members (and is automatically labeled "ID"). Also required is a column labeled DV (dependent variable):

·       Check "Standard Item" and choose "DV".

·       Click "Enter" to add DV as the column name for Column 2.

Click Next> to continue.

 

(The Standard Item menu provides many of the standard data names (DV, TIME, DATE, RATE, AMT, …). If you would like to use another name for one of the standard items in your equations, enter that in the Alias box and click Enter.)
