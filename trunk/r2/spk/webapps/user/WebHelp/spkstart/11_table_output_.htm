11. Table Output.

In this screen, choose values to be output to a table(s).

·       Choose an output file name (RailOutput.txt) and

·       Check "One Header" to get the full output file.

·       Click on "Make Selection" to pop open an Item List.

·       Choose DV from the menu and click Add.

·       Choose also Pred, F, Y and ID and Add each to the List of output items.

·       Then, click on ID (in the output item list) and the ">" button to add "ID" to the List of sorting orders. Similarly, add "DV" to the List.

·       Click "OK" to close the pop-up and

·       then click "Add" on main screen to submit the information.

 

Click Next> to continue.

 

(If additional tables are desired, click "Make Selection" again and repeat the above instructions.)
