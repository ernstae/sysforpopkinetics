Spk-Supported Expression Language

This document lists those types of data, operations and functions which SPK Compiler supports and maps these to equivalent components in C++/Spk.

Data Types and Data Structures

The following data types shall be supported with the limitations laid by the client.
  • natural numbers
  • real numbers
  • negative numbers
  • logical (true/false) values
The following data structures shall be supported on any of the above SPK-supported data types with the limitations laid by the client.
  • scalar
  • vector
  • matrix

Assignment Operator

The assignment operation shall be supported on any combination of scalar, vector and matrix with the restrictions laid by the client.

Boolean Operators

The following types of logical operations shall be supported on any combination of scalar, vector and matrix with the restrictions laid by the client.
  • Determine whether two quantities are equal
  • Determine whether two quantities are different
  • Determine whether one quantity is less than the other
  • Determine whether one quantity is less than or equal to the other
  • Determine whether one quantity is greater than the other
  • Determine whether one quantity is greater than or equal to the other

Control Statement

Control statements shall be supported (ex. if/switch/while/do-while/goto. in C) with the restrictions laid by the client.

Arithmatic Operators / Functions

The following arithmatic operations shall be supported on any combination of scalar, vector and matrix data type with the restrictions laid by the client.
  • addition
  • subtraction
  • multiplication
  • matrix multiplication
  • division
  • matrix division (or back division)
  • modulo
  • absolute value
  • square root
  • power
  • exponential
  • natural log
  • base 10 log
  • cosine
  • sine
  • tangent
  • arc cosine
  • arc sine
  • arc tangent
