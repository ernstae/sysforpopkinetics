12. Plot Output

In this screen, choose values to be plotted.

 

On opening, Plot Data From "1" to "18" should be filled in automatically (representing all 18 data entries).

 

 

To plot modeled (PRED) vs observed (DV) travel times: click on "Make Selection" to pop open an Item List.

On Item list pop-up:

 

  • Select "List 1", find "PRED" in the drop-down menu and click "Add".
  • Select "List 2", find "DV" in the drop-down menu and click "Add."

 

 

Click Next> to continue.

 

(If additional plots are desired, click "Make Selection" again and choose items to plot as before.)
