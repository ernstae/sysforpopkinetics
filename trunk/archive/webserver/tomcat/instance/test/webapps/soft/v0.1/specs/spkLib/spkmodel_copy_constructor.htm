Description
The copy constructor.

Since SpkModel is an abstract class, the direct use of this constructor is only allowed for its subclasses.

The copy constructor is provided to make the SpkModel class more standard even though SPK itself does not currently require its model classes to be copyable.

Arguments
right
The SpkModel to be copied.
