Russian Cyrillic Character Entities (%isocyr1;)

%isocyr1; — Russian Cyrillic Character Entities

Synopsis

The %isocyr1; parameter entity includes the ISO character entities with the public identifier:

  ISO 8879:1986//ENTITIES Russian Cyrillic//EN
