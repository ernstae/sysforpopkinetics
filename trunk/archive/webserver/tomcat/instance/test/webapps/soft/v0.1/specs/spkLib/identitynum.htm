Get a n by n identity matrix
Prototype: const DoubleMatrix identity( int n)

Copyright (C) 2002, University of Washington, Resource Facility for Population Kinetics. All Rights Reserved.

Description
Returns a n by n identity matrix, where n > 0. For n <=0, the program terminates.

Arguments

n
The dimension of the identity matrix.

Example
If you compile, link, and run the following program:

    #include "DoubleMatrix.h"
    #include "identity.h"

    void main(){
        // get a 3 by 3 identity matrix
        DoubleMatrix ident3 = identity(3);

        // display the matrix
        ident3.print();
    }
 then it will display the following when it is run:

[ 1  0  0 ]
[ 0  1  0 ]
[ 0  0  1 ]
