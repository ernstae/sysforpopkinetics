Description
Evaluates the determinant of the square matrix A.

To be specific, this function computes the determinant of A as
                    c
    det( A ) = b * 2  ,

where b is a real number and c is an integer.

Arguments
A
The valarray A contains the square matrix A in the column major order. If the matrix is not LU-decomposable, it throws an SpkException .

n
The integer n% specifies the number of columns in A.

pdB
On output, the double value pointed to by pdB will be equal to b. Note that the double value pointed to by pdB must be declared in the function that calls this function.

plC
On output, the long int value pointed to by plC will be equal to c. Note that the long int value pointed to by plC must be declared in the function that calls this function.

Example
If you compile and link the C++ program,

#include "det.h"
#include "SpkValarray.h"
#include <iostream>
#include <cstdlib>
#include <iomanip>
#include <cmath>

using SPK_VA::valarray;

int main()
{
  using namespace std;

  int nARow = 2;
  int nACol= nARow;
  valarray<double> A( nARow * nACol );

  A[0 + 0 * nARow] = 1000000.0;
  A[0 + 1 * nARow] =      -1.0;
  A[1 + 0 * nARow] =      -1.0;
  A[1 + 1 * nARow] =       2.0;

  double b;
  long int c;
  det( A, nACol, &b, &c );
  
  cout << setiosflags(ios::scientific) << setprecision(15);
  cout << "A = " << endl;
  printInMatrix( A, nACol );
  cout << endl;
  cout << "det( A )     = " << b * pow( 2.0, c ) << endl;

  return EXIT_SUCCESS;
}
