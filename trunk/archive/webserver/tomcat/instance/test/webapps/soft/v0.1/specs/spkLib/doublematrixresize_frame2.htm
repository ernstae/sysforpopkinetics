Description
Changes the dimensions of the matrix. nr and nc must be greater than or equal to zero.

If the product of nr and nc is different from of the original, it will re-allocate the data memory field.

If there is no enough resource available in the heap, it throws SpkException of which the last SpkError is set to SPK_INSUFFICIENT_MEM_ERR.
