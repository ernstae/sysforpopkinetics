Element <table>

Information to request to produce a table. The table columns are placed in the order of which the column appears. There are as many as 10 tables.

Attributes

save_as IMPLIED

The file name in which the table is saved, default to empty which requests display only.

header REQUIRED

The frequency in which the header (ie. labels) appears. "one" requests the header to be appeared only once in the first row for the entire table. "none" requests no header at all. "every" requests a header block per segment, where a segment is composed of 900 records.
