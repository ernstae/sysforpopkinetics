Description
The assignment operator.

Since Covariance is an abstract class, the direct use of this operator is only allowed for its subclasses.

The assignment operator is provide to make the Covariance class more standard even though SPK itself does not currently require Covariance objects to be assignable.

Arguments
right
The Covariance to be copied.
