Creates an Valarray Object Out Of Itself
Prototype: const valarray<double> DoubleMatrix::ToValarray() const

Copyright (C) 2002, University of Washington, Resource Facility for Population Kinetics. All Rights Reserved.

Description
This member function creates and return a valarray object out of itself. The size of the returned array is the product of nr and nc of the calling DoubleMatrix object. The element values of the returned array are copied from the element values of the calling DoubleMatrix object in column-major order. If the size of the calling DoubleMatrix object is zero, the size of the returned array is also zero.

Example
If you compile and link the following program:

  #include "DoubleMatrix.h"
  #include "SpkValarray.h"
  #include <iostream>

  using namespace std;
  void main()
  {
    int i;

    //
    // mat3x2 = [ 0   3 ]
    //          [ 1   4 ]
    //          [ 2   5 ]
    //
    DoubleMatrix mat3x2(3,2);
    for( i=0; i<3*2; i++ )
      mat3x2.data()[i] = i;

    valarray<double> vec = mat3x2.toValarray();
    cout << "valarray = " << endl;
    for( i=0; i<vec.size(); i++ )
    {
      cout << vec[i] << ", ";
    }
    cout << endl;

    cout << "original matrix = " << mat3x2 << endl;
  }
 It produces the following result:
    valarray = 
    0, 1, 2, 3, 4, 5

    original matrix =
    [  0   3  ]
    [  1   4  ]
    [  2   5  ]
