Element <row>

A row in the table/matrix.

Attributes

position IMPLIED

The position of the row in the table/matrix, >= 1. "1" means the row is the first row in the table/matrix. If any row in the table/matrix had this attribute setting, all other rows must have the setting consistently. If no row had this setting, the order of appearance of rows is taken as the order of rows.
