Description
Returns the index to the element that has the largest value in a m by n matrix. The returned index points to the element whose position is computed by i+j*m, for 0 < i < m and 0 < j < n. The routine scans the matrix in the column order and returns the index to the element it finds first.

Arguments

&A
m by n arbitrary matrix .

Example
If you compile, link, and run the following program:

    #include <iostream>
    #include "inxToMax.h"

    void main()
    {
        using namespace std;

        DoubleMatrix dmatA(3,3);
        double *pdA = dmatA.data();

        // set the matrix to:
        // 
        // [0, 3, 2]
        // [1, 0, 3]
        // [2, 1, 0]
        //
        for( int i=0; i<3*3; i++ )
            pdA[i] = i%4;

        cout << inxToMax(dmatA) << endl;

    }

 then it will display the following when it is run:

    Index to max: 3
