Print Array In Matrix Form
Prototype: template <class /T/> void printInMatrix( const SPK_VA::valarray<T>& a, int nCols )
See also:

Copyright (C) 2002, University of Washington, Resource Facility for Population Kinetics. All Rights Reserved.

Description
Print out an array in a matrix format to the standard output. The array is read in the column-major order.

Arguments

T (Optional)
is a data type of the elements in the array, a. The data type must have cout defined to print out an element.

a
is an SPK_VA::valarray object containing an array of elements of type T.
nCols
specifies the number of columns in the resulting matrix. The value must be greater than zero unless the size of the array is zero. The size of the array must be divisible by nCols. Otherwise, it will terminate the program.

Example
If you compile, link, and run the following program:

   #include <iostream>
   #include "SpkValarray.h"
   #include "printInMatrix.h"

   void main()
   {
      using SPK_VA::valarray;
      using namespace std;

      double data[] = { 1, 2, 3, 4, 5, 6, 7, 8 };
      valarray<double> array( data, 8 );

      printInMatrix( array, 2 );
   }
 then it will display the following when it is run:

[ 1 3 5 7 ]
[ 2 4 6 8 ]
