Element <pop_obj_grid>

The data in this element is the integral approximation corresponding to evaluation of the integrand on a uniform grid.

Note that this approximation is not restricted to the Linear Model case as is the pop_obj_analytic calculation; i.e., pop_obj_grid is a valid approximation for all models. An element of this type is present if and only if the integral approximation is requested and a pop_monte_result element is present.
