Transpose of Matrix
Prototype: const SPK_VA::valarray<double> transpose( const SPK_VA::valarray<double> &a, int &n )
See also: DoubleMatrix version .

Copyright (C) 2002, University of Washington, Resource Facility for Population Kinetics. All Rights Reserved.

Description
Returns the transpose of a matrix.

Return Value
Given a m by n matrix A, this function returns an array containing the values of the transpose of A, a n by m matrix B in column-major order.

Arguments
a
is an array containing the value of a m by n matrix, A, in column-major order, where m and n are greater than or equal to zero.

n
is the number of columns in the matrix contained in the array A. The value must be greater than zero if A is not empty.

Example
If you compile, link, and run the following program,

#include "SpkValarray.h"
#include "transpose.h"

void main()
{
  using SPK_VA::valarray;
  using namespace std;
  
  int i;

  // Set A to a 3 by 2 matrix:
  //     /        \
  // A = |  1  4  |
  //     |  2  5  |
  //     |  3  6  |
  //     \        /
  //
  double a[] = { 1, 2, 3, 4, 5, 6 };
  valarray<double> A(a, 3 * 2);

  // Compute:
  //      /           \
  // A' = |  1  2  3  |
  //      |  4  5  6  |
  //      \           /
  //
  valarray<double> aT = transpose(a, 2);   // 2 is the number of column in A.
  cout << "A^T = " << aT << endl;
}
 a vector:
   A^T = {1, 4, 2, 5, 3, 6 }
 will be printed.
