Event Identification Data Item (EVID)

The event identification (EVID) data item is used to indicate which column in the data file contains the EVID values. These values describe the kind of event: observation, dose, other-type, reset, or reset-and-dose. The only value in the EVID column that will have any special meaning within SPK is EVID = 0, which corresponds to observation events.

Note that if every event is an observation event, then the EVID column can be left out of the data file and the EVID label does not need to be included in the $INPUT record.
