Non-parametric Population Analysis Spk Model Requirements

Syntax
SpkModel<Scalar> model.

Purpose
This section documents the minimal requirements for an Spk model to function with the routine Spk non-parameter population analysis routine spk_non_par .

Notation

Scalar
The type Scalar can be any type that supports all the operations supported by double and the CppAD::AD template type.

m
number of fixed effects.

n
number of random effects per individual. (Each individual has it's own realization of the random effects vector.)

M
number of individuals in the population study.

i
the index corresponding to the current individual,  0 \leq i < M

alpha
value of fixed effects,  \alpha \in \R^m .

b
value of random effects for one individual,  b \in \R^n

N[i]
number of measurements for i-th individual.

y^i
measurement vector for i-th individual,  y^i \in \R^{N[i]} .

D(alpha)
covariance of random effects as a function of the fixed effects,  D : \R^m \rightarrow \R^{n \times n} . (These matrices are always symmetric positive definite matrix.)

R(i,alpha,b)
covariance of measurement noise as a function of the individual index, the fixed effects, and the random effects.  R : \B{Z}_+ \times \R^n \times \R^m \rightarrow \R^{N[i] \times N[i]} . (These matrices are always symmetric positive definite matrix.)

f(i,alpha,b)
mean of measurement as a function of the individual index, the fixed effects, and the random effects. given random effects,  f : \B{Z}_+ \times \R^n \times \R^m \rightarrow \R^{N[i]} .

Setting Model State

setPopPar
The syntax
     model.setPopPar  (alpha)
sets the value of the fixed effect vector alpha for subsequent usage of model. The argument alpha must have prototype
     const valarray<Scalar> &alpha
and size equal to m . Calling setPopPar erases any previous setting by the selectIndividual and setIndPar .

selectIndividual
The syntax
     model.selectIndividual  (individual)
sets the index for the individual that corresponds to subsequent usage of model; i.e., i . The argument individual must be an int that is greater than or equal zero and less than M . Calling selectIndividual erases any previous setting by the setIndPar .

setIndPar
The syntax
     model.setIndPar  (b)
sets the value of the random effects b (corresponding to the current individual as specified by selectIndividual ) during subsequent usage of model. The argument b must have prototype
     const valarray<Scalar> &b
and size equal to n .

dataMean
The syntax
     model.dataMean  (f_out) 
sets f_out equal to f(i,alpha,b) . The argument f_out has prototype
     valarray<Scalar> &f_out
and its input and out size equal is  N[i] . The input values of its elements do not matter and the output value is the mean of y^i given the current fixed and random effects.

dataVariance
The syntax
     model.dataVariance  (R_out) 
sets R_out equal to R(i,alpha,b) . The argument R_out has prototype
     valarray<Scalar> &R_out
and its input and out size equal is  N[i] \times N[i] . The input values of its elements do not matter and the output value is the variance of y^i given the current fixed and random effects.
