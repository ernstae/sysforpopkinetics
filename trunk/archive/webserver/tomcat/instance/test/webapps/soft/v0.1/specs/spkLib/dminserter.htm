Inserter <<
Prototype ostream& DoubleMatrix::operator<<(ostream& os, const DoubleMatrix& ob)

Copyright (C) 2002, University of Washington, Resource Facility for Population Kinetics. All Rights Reserved.

Description
The operator inserts a DoubleMatrix object, ob, to the left hand side ostream&, os, in the format shown below. If the matrix is empty, it inserts nothing. For a 3 by 2 matrix, A, cout << "A = " << A; produces:
     3 by 2
     [ a11  a12 ]
     [ a21  a22 ]
     [ a31  a32 ]
 If A were empty, then the same code produces:
     0 by 0
     [ ]
