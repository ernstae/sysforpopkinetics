Non-Russian Cyrillic Character Entities (%isocyr2;)

%isocyr2; — Non-Russian Cyrillic Character Entities

Synopsis

The %isocyr2; parameter entity includes the ISO character entities with the public identifier:

  ISO 8879:1986//ENTITIES Non-Russian Cyrillic//EN

The following character entities are defined in this entity set:
