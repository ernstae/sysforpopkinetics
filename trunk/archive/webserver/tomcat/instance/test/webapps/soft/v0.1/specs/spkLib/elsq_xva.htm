Derivative of elsq With Full Covariance (valarray version)
syntax const valarray<double> elsq_x(
    const valarray<double> &z, 
    const valarray<double> &h, 
    const valarray<double> &Q, 
    const valarray<double> &Qinv
    const valarray<double> &h_x
    const valarray<double> &Q_x)
See also elsq .

Description
The routine elsq_x computes the 1st derivative of extended least squares function (elsq) with respect to x. \ If g(x) is the extended least squares function, then (dg/dx) is defined by
              1               T      -1
    g_x(x) = --- { rvec[Q(x)] }  Q_x(x)
              2  
                       T  -1
              -[z - h(x)]  Q  h_x(x)
                 
               1           T             T        -1
              +--- { [z-h(x)] kron [z-h(x)] } Q_x(x)
               2 
x and z are n dimensional column vectors, Q(x) is a symmetric, positive definite matrix valued function, and h(x) is a column vector valued function. The routine elsq_x computes the derivative of g(x); i.e., g_x(x).

Reference
See Lemma 11 of Approximating the maximum likelihood estimate for models with random parameters.

Return Value
Upon a successful completion, this function returns the g_x(x) value as a column vector, where i-th element of the vector is the derivative of g(x) with respect to x[i].

Arguments
Given a n dimensional column vector, x,
z
This is a valarray<double> object containing the value of z matrix in the column order. z is a n dimensional column vector.
h
This is a valarray<double> object containing the value of h(x). h(x) is a vector valued function, yielding in n dimensions.
Q
This is a valarray<double> object containing the value of Q(x). Q(x) is a matrix valued function, yielding n by n dimensions. The value of Q() is expected to have symmetric, positive definite properties.
Qinv
This is a valarray<double> object containing the value of Q(x)^(-1).
h_x
This is a valarray<double> object containing the value of derivative of h(x); i.e., h_x(x). The derivative yields in n by nX dimensions, where nX is the length of x parameter vector.
Q_x
This is a valarray<double> object containing the value of derivative of Q(x); i.e., Q_x(x). The derivative yields in n^2 by nX dimensions, where nX is the length of x parameter vector.

Example
Suppose that
              / 2 x(1)     0  \            / x(3) \        / 0 \
       Q(x) = |               |    h(x) =  |      |    z = |   |
              \  0      x(2)  /            \ x(3) /        \ 0 /
It follows that                              
    g(x) = 1/2 log[ (2 pi)^2 2 x(1) x(2)] + 1/4 x(3)^2 [1 / x(1) + 2 / x(2)]
The gradient of g(x) is equal to
     1  /  2 / x(1) - x(3)^2 / x(1)^2    \
    --- |  2 / x(2) - 2 x(3)^2 / x(2)^2  |
     4  \ 2 x(3) [ 1 / x(1) + 2 / x(2)]  /
If all the components of x are one, the derivative of g(x) is equal to
    [ 1 , 0 , 6 ] / 4 = [ .025 , 0 , 1.5 ]
