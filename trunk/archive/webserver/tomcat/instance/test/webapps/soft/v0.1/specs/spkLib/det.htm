Determinant of a Positive Definite Symmetric Matrix
Prototype void det( const DoubleMatrix& dmatA, double* pdB, long int* plC )
See also: inverse()

Copyright (C) 2002, University of Washington, Resource Facility for Population Kinetics. All Rights Reserved.

Description
Evaluates the determinant of the positive definite symmetric matrix A.

To be specific, this function computes the determinant of positive definite symmetric A as
                    c
    det( A ) = b * 2  ,

where b is a real number and c is an integer.

Arguments
dmatA
The DoubleMatrix dmatA contains the positive definite symmetric matrix A. If the matrix does not have these properties, it throws an SpkException .

pdB
On output, the double value pointed to by pdB will be equal to b. Note that the double value pointed to by pdB must be declared in the function that calls this function.

plC
On output, the long int value pointed to by plC will be equal to c. Note that the long int value pointed to by plC must be declared in the function that calls this function.

Example
If you compile and link the C++ program,

#include "det.h"
#include "DoubleMatrix.h"
#include <iostream>
#include <cstdlib>
#include <iomanip>
#include <cmath>

int main()
{
  using namespace std;

  int nARow = 2;
  int nACol= nARow;
  DoubleMatrix dmatA( nARow, nACol );
  double* pdAData = dmatA.data();
  pdAData[0 + 0 * nARow] = 1000000.0;
  pdAData[0 + 1 * nARow] =      -1.0;
  pdAData[1 + 0 * nARow] =      -1.0;
  pdAData[1 + 1 * nARow] =       2.0;

  double b;
  long int c;
  det( dmatA , &b, &c );
  
  cout << setiosflags(ios::scientific) << setprecision(15);
  cout << "A = " << endl;
  dmatA.print();
  cout << endl;
  cout << "det( A )     = " << b * pow( 2.0, c ) << endl;

  return EXIT_SUCCESS;
}

 then it should output the following when it is run:

[-1.000000000000000e+000, 2.000000000000000e+000]
