Check whether matrix is empty
Prototype bool DoubleMatrix::isEmpty()

Copyright (C) 2002, University of Washington, Resource Facility for Population Kinetics. All Rights Reserved.

Description
Returns true if the matrix is empty, where "empty" is defined as a matrix whose either or both dimension(s) is zero . Otherwise returns false.

Example
If you compile, link and run the following program
#include <iostream>
#include "DoubleMatrix.h"

void main(){

     using namespace std;
     DoubleMatrix A;
     cout << "A is " << (A.isEmpty()? "empty" : "not empty") << "." << endl;
     }

 The following text appears.  
     A is empty.

will be printed.
