Element <omega_in>

A minimal representation of the initial value of Omega matrix

For full (ie. "block") matrix, omega_in lists the lower triangle elements in the row-major order. For diagonal, omega_in lists only the diagonal elements from the upper left corner. For block diagonal, each omega_in block is listed individually using full or diagonal specifications as appropriate. Given a 3 by 3 matrix, A: 

/                   \
|  a11   a12   a13  |
|  a21   a22   a23  |
|  a31   a32   a33  |
\                   /
If A were full (ie. block) symmetric, omega_in specifies as follows: 
    <omega_in>
       <value>a11</value>
       <value>a21</value><value>a22</value>
       <value>a31</value><value>a32</value><value>a33</value>
    </omega_in>
If A were diagonal, omega_in specifies as follows: 
    <omega_in>
       <value>a11</value>
                         <value>a22</value>
                                           <value>a33</value>
    </omega_in>
If A were block diagonal symmetric with a 2x2 full block and a 1x1 diagonal block, omega_in specifies as two blocks as follows: 
    <omega_in>
       <value>a11</value>
       <value>a21</value><value>a22</value>
    </omega_in>
    <omega_in>
       <value>a33</value>
    </omega_in>

Attributes

same_as_previous IMPLIED

If "yes", constrains the block equal to the previous block of a block diagaonal Omega matrix.

dimension REQUIRED

The dimension of the Omega matrix.

struct REQUIRED

The structure of the Omega matrix: "block" or "diagonal".
