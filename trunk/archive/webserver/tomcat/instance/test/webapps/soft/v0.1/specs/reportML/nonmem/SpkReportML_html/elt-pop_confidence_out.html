Element <pop_confidence_out>

The confidence interval for the final estimate of population parameter.

The first n <value>s are the lower bounds and the remaining n <value>s are the upper bounds, where n is the length of parameter vector. Thus, the lower bound for the i-th element of the parameter vector is found at (0 * n + i)-th in the <value>s sequence and the upper bound for the same parameter is found at (1 * n + i)-th.
