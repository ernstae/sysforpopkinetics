Description
The public interface void cov_popPar( DoubleMatrix& \ret\ ) const evaluates the user defined virtual doCov_popPar( DoubleMatrix& \ret\ ) const at the current evaluation points.

Arguments
ret 
is n by nPopPar matrix resulted from evaluating doCov_popPar(), where n is the product of #rows and #cols of a matrix resulted from evaluating doCov() and nPopPar is the size of the population parameter vector. j-th column of matrix, ret, contains the derivative of the covariance with respect to j-th element of the population parameter vector, in the column-major order. Each column, or the derivative with respect to j-th, must be a symmetric positive definite matrix.
