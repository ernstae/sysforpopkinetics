Element <model>

Definition of NONMEM mixed effect model.

  • All variables are treated as double-precision scalars. The data type of contant (literal) values retain as it is.
  • Array specification is not supported.
  • Cases are ignored.
  • The following unary functions are supported:
    • ATAN ( real )
    • COS ( real )
    • EXP ( real )
    • LOG ( real )
    • LOG10( real )
    • SIN ( real )
    • SQRT ( real )
    • ABS ( real )
    • ACOS ( real )
    • ASIN ( real )
    • ATAN2( real )
    • COSH ( real )
    • MAX ( real )
    • MIN ( real )
    • MOD ( real )
    • SINH ( real )
    • TAN ( real )
    • TANH ( real )
    • LININTERP()
The valid combinations of PREDPP library model and TRANS:
Advan1: One compartmental linear model
Trans1, Trans2
Advan2: One compartmental linear model with first order absorption
Trans1, Trans2
Advan3: Two compartmental linear model
Trans1, Trans3, Trans4, Trans5
Advan4: Two compartmental linear model with first order absorption
Trans1, Trans3, Trans4, Trans5
Advan5: General linear model
Trans1
Advan6: General nonlinear model
Trans1
Advan7: General linear model with real eigenvalues
Trans1
Advan8: General nonlinera model with stiff differential equations
Trans1
Advan9: General nonlinear model with equilibrium compartments
Trans1
Advan10: One compartmental model with Michaelis-Menten elimination
Trans1

Attributes

tolerance IMPLIED

Number of digits of accuracy, used when an ADVAN model is selected.

advan IMPLIED

The number that corresponds to PREDPP library model from ADVAN1 - ADVAN10. Currently supported models are { 6 }. If this attribute is omitted, pred must be defined. The attribute setting will be ignored if pred is specified.

trans IMPLIED

The number that corresponds to parameterization for the selected ADVAN model. Choose from {1,2,3,4,5}, default to "1". If no ADVAN model is used, this setting is ignored.
