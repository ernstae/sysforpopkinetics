Description
This function evaluates the derivative of the equivalent data variance with respect to the population parameter vector of the SpkModel class object at a currently set population parameters of the original SpkModel class object and at zero value of individual parameters of the original SpkModel class object.
The evaluation result of doDataMean_indPar() shall be placed in R_b. It returns true if f_b contains at least one non-zero value. If all zeros, it returns false.

Arguments
f
shall, upon a successful completion of evaluation, contain a resulting matrix of size nY*nY by nAlp, where nY is the number of all individual's data and nAlp is the number of population parameters of the original SpkModel class object in the following form.

                  /  ~                   ~             T                                                              \
                  |  d  f1(alp, 0)D(alp)[d  f1(alp, 0)]  + R1(alp, 0)                   0                             |
                  |   b1                  b1                                                                          |
                  |                                              .                                                    |
R_alp(alp) = d    |                                                 .                                                 |
              alp |                                                    .                                              |
                  |                                                  ~                   ~             T              |
                  |                0                                 d  fn(alp, 0)D(alp)[d  fn(alp, 0)]  + Rn(alp, 0) |
                  \                                                   bn                  bn                          /

where f1,...,fn are the data means of individual 1,...,n of the original SpkModel class object. b1,...,bn are the individual parameters of individual 1,...,n of the original SpkModel class object. alp is the population parameter vector of the original SpkModel class object and n is the number of individuals of the original SpkModel class object. R1,...,Rn are data variances of individual 1,...,n of the original SpkModel class object and D is the individual parameter variance of the original SpkModel class object. Symbol ~ indicates taking central difference.
