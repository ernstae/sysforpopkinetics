Element <simulation>

This element appears only if data simulation is requested.

It is empty but carries a few attribute values such as the seed used to simulate a data set.

Attributes

seed REQUIRED

The seed value used to simulate the data set.

subproblem REQUIRED

The iteration number during which this particular simulation was done. The value, 1, indicates the first iteration.

Content
