Description
The routine elsq_x computes the 1st derivative of extended least squares function (elsq) with respect to x. \ If g(x) is the extended least squares function, then (dg/dx) is defined by
              1               T      -1
    g_x(x) = --- { rvec[Q(x)] }  Q_x(x)
              2  
                       T  -1
              -[z - h(x)]  Q  h_x(x)
                 
               1           T             T        -1
              +--- { [z-h(x)] kron [z-h(x)] } Q_x(x)
               2 
x and z are column vectors, Q(x) is a symmetric, positive definite matrix valued function, and h(x) is a column vector valued function. The routine elsq_x computes the derivative of g(x); i.e., g_x(x).

Reference
See Lemma 11 of Approximating the maximum likelihood estimate for models with random parameters.

Return Value
Upon a successful completion, this function returns the g_x(x) value as a column vector, where i-th element of the vector is the derivative of g(x) with respect to x[i].

Arguments
Given a n dimensional column vector, x,
z
is a m by 1 double-precision matrix.
h
is a m by 1 double-precision matrix containing the value of h(x).
r
is a m by 1 double-precision matrix containing the value of z - h(x).
Q
is a m by m double-precision containing the value of Q(x).
Qinv
is a m by m double-precision matrix containing the matrix inverse of Q.
h_x
is a m by n double-precision matrix containing the derivative of h(x); i.e., h_x(x).
Q_x
is a m*m by n double-precision matrix containing the derivative of Q(x); i.e., Q_x(x).

Example
Suppose that
              / 2 x(1)     0  \            / x(3) \        / 0 \
       Q(x) = |               |    h(x) =  |      |    z = |   |
              \  0      x(2)  /            \ x(3) /        \ 0 /
It follows that                              
    g(x) = 1/2 log[ (2 pi)^2 2 x(1) x(2)] + 1/4 x(3)^2 [1 / x(1) + 2 / x(2)]
The gradient of g(x) is equal to
     1  /  2 / x(1) - x(3)^2 / x(1)^2    \
    --- |  2 / x(2) - 2 x(3)^2 / x(2)^2  |
     4  \ 2 x(3) [ 1 / x(1) + 2 / x(2)]  /
If all the components of x are one, the derivative of g(x) is equal to
    [ 1 , 0 , 6 ] / 4 = [ .025 , 0 , 1.5 ]
