Get a pointer to the first element in a matrix
Prototype *double DoubleMatrix::data()

Copyright (C) 2002, University of Washington, Resource Facility for Population Kinetics. All Rights Reserved.

Description
data() returns the pointer to the first element of the matrix. When the matrix is empty, a null is returned. Note that after an assignment operator is called or after the matrix is passed into a function without const, the pointer becomes invalid. Therefore, it is recommended that calling data() to return the pointer immediately before it is used.

UnitTest Plans
Check things work for a 0x0, 0x1, 0xn, 1x0, 1xn, nxn matrix.

Example
If you compile, link and run the following program

#include <iostream>
#include "DoubleMatrix.h"
using namespace std;

void main(){

     DoubleMatrix A(3, 2);
     double *pA = A.data();

     // Initialize A to a matrix
     // [ 1  4 ]
     // [ 2  5 ]
     // [ 3  6 ]
     for( int i=0; i<6; i++ )
          pA[i] = i+1;

     A.print();
}

 a matrix  
     [ 1  4 ]
     [ 2  5 ]
     [ 3  6 ]

will be printed.
