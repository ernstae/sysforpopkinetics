Description
This member function returns a (size of array)/(nc) by nc matrix. The element values of the matrix are copied from the element values of the valarray object, array, in column-major order. If the size of the valarray object is zero, the function returns a zero size DoubleMatrix object. The product of #rows and #cols of the original DoubleMatrix matrix does not have to match the size of array.

If there is no enough resource available in the heap, it throws SpkException of which the last SpkError is set to SPK_INSUFFICIENT_MEM_ERR.

UnitTest Plans
Check if a (size of array)/(nc) by nc matrix is successfully returned from this function. Check if a zero size matrix can be successfully returned from this function..
