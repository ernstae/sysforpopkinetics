Element <sigma>

A minimal representation of NONMEM Omega matrix. For full (ie. "block") matrix, sigma lists the lower triangle elements in the row-major order. For diagonal, sigma lists only the diagonal elements from the upper left corner. For block diagonal, each sigma block is listed individually using full or diagonal specifications as appropriate.

Given a 3 by 3 matrix, A: 

/                   \
|  a11   a12   a13  |
|  a21   a22   a23  |
|  a31   a32   a33  |
\                   /
If A were full (ie. block) symmetric, sigma specifies as follows: 
    <sigma>
       <value>a11</value>
       <value>a21</value><value>a22</value>
       <value>a31</value><value>a32</value><value>a33</value>
    </sigma>
If A were diagonal, omega specifies as follows: 
    <sigma>
       <value>a11</value>
                         <value>a22</value>
                                           <value>a33</value>
    </sigma>
If A were block diagonal symmetric with a 2x2 full block and a 1x1 diagonal block, sigma specifies as two blocks as follows: 
    <sigma>
       <value>a11</value>
       <value>a21</value><value>a22</value>
    </sigma>
    <sigma>
       <value>a33</value>
    </sigma>

Attributes

same_as_previous IMPLIED

"yes" to duplicate the initial and final values of the previously appeared Sigma block's values to this block's, default to "no".

struct REQUIRED

Structure of Sigma matrix, either "block" or "diagonal". "block" suggests that any elements in the symmetric matrix could be non-zero. "diagonal" indicates all zeros except for the diagonal elements.

dimension REQUIRED

Dimension of the square Sigma matrix. For example, for a 3 by 3, say "3".
