Description
Writes to the restart file the name and value of an attribute that is contained in the current element's start tag, i.e., <current_element attribute_name="########"> where the sequence of number signs (#) represents the sequence of characters that constitute the attribute's value.

This function assumes that the current element start tag except for its final angle bracket (>) has been written to the restart file before this function is called and that its final angle bracket will be written after.

Arguments
ValType
The type of the value.

pAttributeName
This argument specifies the name of the attribute to write.

valueIn
This argument will be written as the value of the attribute using the std::ostream insertion operator (<<), which converts it from type ValueType to a sequence of characters.
