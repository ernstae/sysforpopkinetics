Element <compartment>

Definition of a compartment.

Attributes

no_dose IMPLIED

"yes" prohibits this compartment from receiving a dose, default to "no".

exclude IMPLIED

"yes" makes this compartment's amount exclueded from the amount of the output compartment (ADVAN9 only), default to "no".

name REQUIRED

Name of the compartment.

default_observation IMPLIED

"yes" makes this compartment the default observation compartment, default to "no" which means the compartment with the name "CENTRAL" or the first compartment.

initial_off IMPLIED

"yes" makes this compartment initially off, default to "no".

no_off IMPLIED

"yes" prohibits this compartment from being turned on or off, default to "no".

default_dose IMPLIED

"yes" makes this compartment the edefault dose compartment, default to "no" which means the compartment with the name "DEPOT" or the first compartment.

equilibrium IMPLIED

"yes" makes this compartment an equilibrium compartment, default to "no".
