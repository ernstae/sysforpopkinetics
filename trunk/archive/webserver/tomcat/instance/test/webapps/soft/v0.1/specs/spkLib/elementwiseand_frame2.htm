Description
Given m by n matrices A and B, Returns a m by n matrix C such that c(i,j) = 1.0 if a(i,j) and b(i,j) are both 1.0; c(i,j) = 0.0 otherwise, where i and j are the row index and the column index, respectively.

If the dimensions do not match, the program terminates.

Arguments

&A
is a m by n matrix.

&B
is another m by n matrix.

Example
If you compile, link, and run the following program:

  #include "DoubleMatrix.h"
  #include "elementwiseAnd.h"

  void main(){
    
    // Two matrices must have the same dimensions  
    DoubleMatrix dmatA(3,3);
    DoubleMatrix dmatB(3,3);

    double *pdA = dmatA.data();
    double *pdB = dmatB.data();

    //
    // Initialize A to a matrix:
    //    [ 0  3  6 ]
    //    [ 1  4  7 ]
    //    [ 2  5  8 ]
    //
    // and B to a matrix:
    //    [ 8  5  2 ]
    //    [ 7  4  1 ]
    //    [ 6  3  0 ]
    //
    for( int i=0; i<9; i++ ){
        pdA[i] = i;
        pdB[i] = 8-i;
    }

    // Take && and print the resulting matrix
    // Note that any value greater than 0 is considered TRUE
    and(dmatB, dmatA).print();
  }

 then it will display the following when it is run:

    [ 0  1  1 ]
    [ 1  1  1 ]
    [ 1  1  0 ]
