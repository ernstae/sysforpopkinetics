Description
Gets the value from a single "value" element that is contained in a subelement of the current element, i.e., <current_element> ... <subelement> <value>########</value> </subelement> ... </current_element> where the sequence of number signs (#) represents the sequence of characters that constitute a single value.

This function assumes that the current element has been set before this function is called.

Arguments
ValType
The type of the value.

pSubelementTag
This argument specifies the name of the subelement of the current element that contains the single "value" element.

valueOut
This argument will be set equal to the value that corresponds to the contents of the "value" element. after it has been converted to the type of ValueType using the std::istream extraction operator (>>).
