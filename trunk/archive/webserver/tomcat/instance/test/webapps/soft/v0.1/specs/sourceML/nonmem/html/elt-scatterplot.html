Element <scatterplot>

Information to request to produce a scatterplot. At most 2 splits are allowed.

Attributes

x0_line IMPLIED

"show" to show a vertical line on x=0, "hide" to hide it, default to "show".

y0_line IMPLIED

"show" to show a horizontal line on y=0, "hide" to hide it, default to "show".

unit_slope IMPLIED

"show" to show the unit slope, "hide" to hide it, default to "hide".

to IMPLIED

The number of the last data record which may contribute to the scatterplot, default to "from"+899.

from IMPLIED

The number of the first data record which may contribute to the scatterplot, default to 1.

Content

(description? | x+ | y+ | split*)
