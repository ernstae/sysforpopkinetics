Description
Required for population analysis.

SpkModel<Scalar>::doDataVariance_popPar() calculates the derivative of the variance of a particular individual's data with respect to the population parameter, for a currently selected individual (see selectIndividual ) at a currently set individual parameter (see setIndPar ) and population parameter (see setPopPar ).

The evaluation result of doDataVariance_popPar() shall be placed in ret. It returns true if ret contains at least one non-zero value. If all zeros, it returns false.

doDataVariance_popPar() shall be only called via its corresponding public interface, dataVariance_popPar(). The public interface shall be called only by the SPK library components unless when it is being unit-tested, or SPK cannot guarantee any of the computational results.

The public interface, doDataVariance_popPar() catches any exception thrown by the private interface, dataVariance_popPar(), implemented by the user. If the exception is an instance of SpkException , an additional SpkError object will be appended to the list of errors maintained within the object. The new SpkError object will have error code set to SpkError::SPK_MODEL_DATA_VARIANCE_POP_ERR.

Reference
B. M. Bell, Approximating The Marginal Likelihood Estimate For Models With Random Parameters, Applied Mathematics and Computation, Accepted 1999.

Arguments
ret
shall, upon a successful completion of evaluation, contain a resulting vector of size nYi * nYi * nPopPar, where nYi is the number of measurements for a currently selected individual and nPopPar is the size of population parameter vector. Let R denotes the function expressed by doDataVariance(). The resulting vector contains a sequence of partial derivatives of R(popPar, indPar) with respect to each element of popPar (population parameter vector):

  ret = { R(popPar, indPar)_popPar(1), R(popPar, indPar)_popPar(2), ... R(popPar, indPar)_popPar(nPopPar) }

where popPar(k) denotes k-th element of popPar vector. R(popPar, indPar)_popPar(k) holds the symmetric, positive definite property.
