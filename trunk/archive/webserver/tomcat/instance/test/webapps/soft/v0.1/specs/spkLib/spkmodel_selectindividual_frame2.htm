Description
Required for population analysis.

doSelectIndividual( int i ) selects i-th individual in the population and guarantees that subsequent attempts for evaluating the mathematical models expressed by SpkModel virtual member functions yield to the corresponding individual's values at the current parameters. The valid value range for i is 0 to N-1, where N is the number of individuals in the population. i=0 shall select the first individual.

doSelectIndividual() shall be only called via its corresponding public interface, selectIndividual(). The public interface shall be called only by the SPK library components unless when it is being unit-tested, or SPK cannot guarantee any of the computational results.

The public interface, selectIndividual() catches any exception thrown by the private interface, doSelectIndividual(), implemented by the user. If the exception is an instance of SpkException , an additional SpkError object will be appended to the list of errors maintained within the object. The new SpkError object will have error code set to SpkError::SPK_MODEL_SET_INDEX_ERR.

Arguments
i
An integer indicating the individual to be selected. The valid value range is from 0 to N-1, where N is the number of individuals in the population. 0 shall point to the first individual in the population.
