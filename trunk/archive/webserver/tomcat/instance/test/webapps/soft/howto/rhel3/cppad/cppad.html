Installation

  1. Download current version of CppAD from the web page
             http://www.coin-or.org/CppAD/Doc/installunix.htm  
    into /tmp on the installation machine.

  2. Extraction 

            cd /tmp
        tar -xvzf CppAD-yy-mm-dd.tar.gz
    where yy is the year, mm is the month, and dd is the day (each as two decimal digits) corresponding to the version of CppAD. This will extract files into a new directory whose name is cppad-yy-mm-dd.

  3. Configure 

            cd cppad-yy-mm-dd
        ./configure --prefix=/usr/local POSTFIX_DIR=spktest

  4. Make 

            make

  5. Final Installation 

            su
        make install
