Description
Writes the start tag for an XML element.

Arguments
pTag
This argument specifies the name of the start tag to write.

writeClosingBracket
If this argument is equal to true, then the closing bracket (>) will be written after the start tag name.

newLineAfter
If this argument is equal to true, then a new line will be written after the start tag.
