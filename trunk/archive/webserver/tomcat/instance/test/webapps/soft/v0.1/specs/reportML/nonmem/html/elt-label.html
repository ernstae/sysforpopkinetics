Element <label>

The name/label of the item.

Attributes

name REQUIRED

A string begining with an alphabet followed by any combination of alphabets, numbers, _ (underscore) that serves as the label. Case-sensitive.

synonym IMPLIED

[optional] Similar to "name" but this serves as an alias/synonym that references the same entity as "name" does, default to "" (none).
