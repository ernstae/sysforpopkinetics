Description
The routine elsq computes the extended least squares function g(x) which is defined by
            1                         1            T  -1
    g(x) = --- logdet[ 2 pi Q(x) ] + --- [z - h(x)]  Q (x) [z - h(x)]
            2                         2
x and z are column vectors, Q(x) is a symmetric, positive definite matrix valued function, and h(x) is a column vector valued function.

Reference
See Lemma 11 of Approximating the maximum likelihood estimate for models with random parameters.

Return Value
Upon a successful completion, this function returns the value g(x) as a double-precision scalar.

Arguments
z
is a m dimensional column vector. In the context of population analysis, this corresponds to the data vector (yi).

h
is a m dimensional column vector containing the value of h(x). It has the same number of rows as z. In the context of population analysis, this corresponds to the vector value returned by evaluating the model for the variance of i-th data at x.

r
is a m dimensional column vector containing the value of z - h(x).

Q
is a m by m square matrix containing the value of Q(x) corresponding to the current value of x. It has the same row dimension as z. In the context of population analysis, this corresponds to the symmetric matrix returned by evaluating the model for the variance of the i-th individual's data at x.

Qinv
is a square matrix containing the matrix inverse of Q. It has the same dimensions a Q.

Example
If you compile, link and run the following program;
    #include <iostream>
    #include <cassert>
    #include "DoubleMatrix.h"
    #include "elsq.h"
    #include "inv2by2.h"
    #include "det.h"

    void main(){

        using namespace std;

        int i;
        DoubleMatrix z(2,1);  // column vector
        DoubleMatrix h(2,1);  // column vector
        DoubleMatrix Q(2,2);  // symmetric and invertible
        DoubleMatrix Qinv(2,2);

        double *pz = z.data();
        double *ph = h.data();
        double *pQ = Q.data();
        double ans;

        // Set Q to a matrix
        //   [ 2.0  1.0 ]
        //   [ 1.0  3.0 ]
        pQ[0] = 2.0;
        pQ[1] = 1.0;
        pQ[2] = 1.0;
        pQ[3] = 3.0;

        // Compute inverse of Q
        Qinv = inverse(Q);

        // Set h and z to a matrix
        //   [ 1.0 ]
        //   [ 1.0 ]
        for( i=0; i<2; i++ ){
            pz[i] = 1.0;
            ph[i] = 1.0;
        }

        cout << "Q = " << endl;
        Q.print();
        
        cout << "Qinv = " << endl;
        Qinv.print();

        // Compute elsq and display the result
        ans = elsq(z, h, Q, Qinv);
        cout << "elsq = " << ans << endl;
 the program will display;
