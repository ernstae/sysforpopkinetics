Description
This function evaluates the derivative of the mean of all individual's data with respect to the population parameter vector of the SpkModel class object at a currently set population parameters of the original SpkModel class object and at zero value of individual parameters of the original SpkModel class object.

The evaluation result of doDataMean_indPar() shall be placed in f_b. It returns true if f_b contains at least one non-zero value. If all zeros, it returns false.

Arguments
f
shall, upon a successful completion of evaluation, contain a resulting matrix of size nY by nAlp, where nY is the number of all individual's data and nAlp is the number of population parameters of the original SpkModel class object, in the following form.

            /  f1_alp(alp, 0)  \
f_b(alp) =  |   :              |
            \  fn_alp(alp, 0)  /

where f1_alp,...,fn_alp are the derivatives of data mean of individual 1,...,n of the original SpkModel class object with respect to alp, respectively. alp is the population parameter vector of the original SpkModel class object and n is the number of individuals of the original SpkModel class object.
