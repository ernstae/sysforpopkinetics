Description
The assignment operator.

Since FullCovariance is an abstract class, the direct use of this operator is only allowed for its subclasses.

The assignment operator is provide to make the FullCovariance class more standard even though SPK itself does not currently require FullCovariance objects to be assignable.

Arguments
right
The FullCovariance to be copied.
