Description
This function gets any error information that is available and formats it in the same form as the SpkError serialize function.

Arguments
headerStr
This string is added to the beginning of the error message before it is formatted.

messageStr
This string will contain the formatted error message. It will be set equal to an empty string if there is no error information.

lineNumber
This is the line number in the file where the error was generated.

fileName
This is the name of the file where the error was generated.
