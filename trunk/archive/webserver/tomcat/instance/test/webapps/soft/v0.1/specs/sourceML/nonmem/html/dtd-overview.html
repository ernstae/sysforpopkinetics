SpkSourceML : MDA -> ASPK Compiler Information Package

This documentation specifies the information passed from MDA to ASPK.

For MDA, it is to both request (ultimately) SPK to perform either individual analysis or population analysis and present the results before the user when the analysis is completed. For ASPK, it is to have SPK Compiler translate the information into an executable and have CSPK compute and return the results.
