Description
Gets the value of an attribute that is contained in the current element, i.e., <current_element attribute_name="########"> where the sequence of number signs (#) represents the sequence of characters that constitute the attribute's value.

This function assumes that the current element has been set before this function is called.

Arguments
ValType
The type of the value.

pAttributeName
This argument specifies the name of the attribute in the current element for which to get the value.

valueOut
This argument will be set equal to the value that corresponds to the attribute's value after it has been converted to the type of ValueType using the std::istream extraction operator (>>).
