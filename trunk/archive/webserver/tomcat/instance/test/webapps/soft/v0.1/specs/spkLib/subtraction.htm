Element-wise subtraction
Prototype: const DoubleMatrix subtract(const DoubleMatrix &A, const DoubleMatrix &B)
void subtract(const DoubleMatrix &A, const DoubleMatrix &B, DoubleMatrix &C)
See also: addition , matrix multiply , matrix back division , element-wise multiply , element-wise division .

Copyright (C) 2002, University of Washington, Resource Facility for Population Kinetics. All Rights Reserved.

Description
This function performs matrix subtraction.

Returned Value
Given a m by n matrix, A, and another matrix with the same dimensions, B, the resulting matrix is also a m by n. The result is returned by value unless the third argument C is given. When C is given, the result is set to the memory location referred by C.

This function checks to see whether a floating-point error has occurred or not at the end of computation. If detected, it will throw an SpkException whose last (error ) element is set to indicate the nature of the error.

Arguments
A
is the m by nleft hand side matrix, where m and n are greater than or equal to zero.

B
is the m by n right hand side matrix, where m and n are greater than or equal to zero.

C
is the m by n right hand side matrix, which will contain the result, if given.

Example
If you compile, link and run the following program,

   #include "subtract.h"

   void main(){
   
       DoubleMatrix A(3, 2);
        DoubleMatrix B(3, 2);
        DoubleMatrix C;

        double *a = A.data();
        double *b = B.data();

       // Set A to a matrix
        //    [ 0  3 ]
        //    [ 1  4 ]
        //    [ 2  5 ]
        // and B to a matrix
        //    [ 1  1 ]
       //    [ 1  1 ]
        //    [ 1  1 ]
       for( int i=0; i<6; i++ ){
            a[i] = i;
             b[i] = 1;
        }

       // Computes C = A - B
       C = subtract(A, B);

       // display C
        C.print();
   }
 a matrix
   [ -1  2 ]
   [  0  3 ]
   [  1  4 ]

 will be displayed.
