Description
This function performs matrix-matrix addition.

Return Value
Given a m by n matrix, A, and m by n matrix, B, it returns another m by n matrix that is the result of the addition of A and B. For the void version, the result is placed at a memory location referenced by & C.

Arguments
A
is a m by n matrix, where m and n are greater than or equal to zero.

B
is a m by n matrix, where m and n are greater than or equal to zero.

C
is a m by n matrix, if given, which will contain the result.

Example
If you compile, link and run the following program,

   #include "add.h"

   void main(){
   
       DoubleMatrix A(3, 2);
        DoubleMatrix B(3, 2);
        DoubleMatrix C;

        double *a = A.data();
        double *b = B.data();

       // Set A to a matrix
        //    [ 0  3 ]
        //    [ 1  4 ]
        //    [ 2  5 ]
        // and B to a matrix
        //    [ 1  1 ]
       //    [ 1  1 ]
        //    [ 1  1 ]
       for( int i=0; i<6; i++ ){
            a[i] = i;
             b[i] = 1;
        }

       // Computes C = A + B
       C = add(A, B);

       // display C
        C.print();
   }
 the following matrix will be displayed.
   [ 1, 4 ]
   [ 2, 5 ]
   [ 3, 6 ]
 will be displayed.
