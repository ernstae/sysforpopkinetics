Description
Returns the sum of squares of the elements of the vector Z weighted by the inverse of the current value of the covariance matrix. To be specific, this function calculates

     T     -1
    Z   Cov    Z  ,

where Cov is the current value of the covariance matrix.

Note
(1.) This function is declared virtual and can be redefined in subclasses of this class.

(2.) If this function is redefined, it's new implementation must update the cached values before it does any computations. See FullCovariance_RedefiningMathFunctions for more.

Arguments
z 
The vector whose elements will be squared and then weighted in the sum.
