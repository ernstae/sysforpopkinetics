Description
This constructor takes four arguments, pModel, N, bStep and nA, to initialize class member data.

Arguments
pModel 
The SpkModel* pModel is a pointer to a SpkModel class instance that is used for population analysis.

N 
The valarray<int> N contains an array of int values, which specifies the number of data measurements for each individual. The number of elements in N must be equal to nInd, the number of individuals in the original SpkModel class object.

bStep 
The valarray<double> bStep contains an array of double values, which specifies the step size used for approximating the derivatives with respect to the individual parameters. The size of bStep is equal to the length of the individual parameter vector of the original SpkModel class object.

nA 
The int nA specifies the number of population parameters of the original SpkModel class object.
