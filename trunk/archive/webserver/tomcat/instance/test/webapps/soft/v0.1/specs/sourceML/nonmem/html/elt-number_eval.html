Element <number_eval>

Specifies the numbers of function evaluations to use for each individual in the population data set.

To be specific, the number of evaluations is the product of the values in values under number_eval. If "method" is equal to "grid", the number of occurences of value under number_eval is equal to the number of random effects (per individual). The i-th value is the number of uniformly space grid points in the i-th random effect. If "method" is not equal to "grid", the number of occurences of value is equal to one; whence, the first occurence of value is the number of function evaluations to use for each individual. The default value for each value is 1000 for "plain" and "miser". 10 for "grid".
