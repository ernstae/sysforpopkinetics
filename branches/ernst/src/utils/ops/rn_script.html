NAME

release_notes -- combine and publish SPK release notes

SYNOPSIS

release_notes

DESCRIPTION

Each candidate for deployment of SPK may be thought of as a minor version. For each candidate, there is a text file, called notes-version, where version is a minor version number such as "0.1.24". Each such file is created by the stage_candidate.pl shell script when a software engineer creates a candidate. Initially, the file contains only the skeleton of a release-notes document. Subsequently, the software engineer uses a text editor to flesh out the file with a description of the changes that the minor version embodies.

The release_notes script is run as part of the deployment process. All release-notes files created since the last deployment are concatenated to create a combined release-notes file for the deployed version.

OPTIONS AND ARGUMENTS

None.

EXIT STATUS

Returns the value 0.
