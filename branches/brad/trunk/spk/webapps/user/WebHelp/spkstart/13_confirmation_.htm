13. Confirmation.

The confirmation pane gives you a chance to view the choices you have made (in NONMEM syntax). If you would like to change any entries, simply use the "Back" button to return to the appropriate screen.

 

Click "Finish" to create the control file.

 

A popup asks: "Do you want to save the NONMEM control file?" It is not necessary to save the file if you plan to submit the job immediately. Once submitted, the control file will be saved as part of the job in the SPK database. Click Yes and save the file or click No. In either case, the Input File Generation tool will close.

 

Return to the MDA. The viewing pane will contain an XML version of the input specifications you have created.
